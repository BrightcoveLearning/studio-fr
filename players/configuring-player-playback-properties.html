Les Relecture Les propriétés peuvent être utilisées pour configurer la façon dont la lecture vidéo est gérée dans le lecteur.

Pour configurer le Relecture propriétés, procédez comme suit :

  1. Cliquez sur Players dans l'en-tête de navigation pour ouvrir le module Players.
  2. Cliquez sur le lien du lecteur à configurer.
  3. Cliquez sur Lecture dans le menu de navigation de gauche.
Note: Toute modification apportée à un lectuer peut prendre jusqu'à 5 minutes pour apparaître dans les lectuers publiés.

Les propriétés de lecture suivantes peuvent être configurées :

  • Lecture automatique de la vidéo lors du chargement du lecteur
  • Jouer en sourdine
  • Précharger la vidéo avant la lecture
  • Activer DRM
  • Propriétés de la playlist

Lecture automatique de la vidéo lors du chargement du lecteur

Les Lecture automatique de la vidéo au chargement du lecteur Ce paramètre contrôle si la vidéo commencera à jouer lorsque le lecteur est chargé. Ce paramètre peut fonctionner différemment selon le navigateur. Pour plus d'informations, consultez Considérations relatives à la lecture automatique.

Jouer en sourdine

Les Jouer en sourdine Ce paramètre contrôle si le son de la vidéo sera coupé.

Précharger la vidéo avant la lecture

Les Précharger la vidéo avant la lecture Ce paramètre contrôle si le lecteur précharge le contenu vidéo avant que l'utilisateur ne lance la lecture.

Note: Pour plus de détails, consultez la section Détails du préchargement du document Options disponibles pour le document Code incorporé avancé (dans la page).

Les options suivantes sont disponibles :

  • Précharger la vidéo (la plus rapide) - Indique que l'ensemble du fichier vidéo a pu être téléchargé, même si l'utilisateur ne regarde pas la vidéo. Du côté positif, si le spectateur lit la vidéo, il peut commencer à jouer immédiatement. Du côté négatif, cette option augmentera la consommation de bande passante à mesure que la précharge/charge se produit même si le spectateur ne joue pas la vidéo.
  • Métadonnées de préchargement: seules les métadonnées vidéo sont préchargées. C'est la valeur par défaut.
  • Ne pas précharger - Aucune vidéo n'est préchargée

Notes :

  • Lorsque vous utilisez l'Insertion d'annonces côté serveur (SSAI), Brightcove recommande de définir preload=none. S'il preload est défini sur auto ou metadata, les premières images de la vidéo le flux sera demandé, ce qui compte l'annonce comme diffusée même si l'utilisateur ne démarre pas la lecture.
  • La valeur du paramètre de préchargement dans Studio remplacera la valeur du paramètre preload valeur d'attribut dans Code intégré avancé (encart).

Activer DRM

Les Activer DRM le paramètre, lorsqu'il est coché, permet au contenu protégé par DRM d'être lu dans le lecteur.

Propriétés de la playlist

Lorsqu'un Type de lectuer de Playlist est sélectionné dans le cadre du Informations sur le lectuer propriétés, certains supplémentaires Relecture les propriétés seront affichées.

  • Ordre de lecture aléatoire - Les vidéos seront lues dans un ordre aléatoire
  • Lecture de la vidéo lorsque cette option est sélectionnée: si vous cliquez sur une miniature dans le menu de la liste de lecture, la vidéo sera lue lorsqu'elle est chargée, même si la vidéo précédente du lecteur a été suspendue
  • Mode de lecture continue - Les vidéos progressent automatiquement et se répètent indéfiniment
  • Compte à rebours vidéo: uniquement activé lorsque le mode de lecture continue est coché
    • Carte suivante (avance automatique) - Une carte suivante est affichée dans le lectuer
    • Écran de fin suivant (compte à rebours) - Un compte à rebours d'écran de fin s'affiche dans le lecteur
