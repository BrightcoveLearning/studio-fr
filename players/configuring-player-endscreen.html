Les lecteurs peuvent être configurés de sorte qu'un écran de fin s'affiche à la fin d'une vidéo. L'écran de fin peut être configuré pour afficher les icônes de relecture et de partage ou peut être personnalisé pour inclure du code HTML. Un écran de fin personnalisé est montré ci-dessous.

Les icônes s'affichent pour autoriser l'utilisateur à revoir la vidéo ou à la partager sur les réseaux sociaux les plus courants.

Remarque: si un écran de fin de lecteur est configuré, il n'apparaîtra pas sur un appareil iOS si le lecteur est en mode plein écran natif.

Pour configurer un écran de fin de lecteur, procédez comme suit :

  1. Cliquez sur Players dans l'en-tête de navigation pour ouvrir le module Players.
  2. Cliquez sur le lien du lecteur à configurer.
  3. Cliquez sur Écrans de fin dans le menu de navigation de gauche.
  4. Sélectionnez le Écran de fin du lectuer option. Les options suivantes sont disponibles :
    • Aucun - Aucun écran de fin ne sera affiché
    • Social: affiche la boîte de dialogue de partage social à la fin de la vidéo
    • Personnalisé: affiche un écran de fin personnalisé à la fin de la vidéo
Note: Toute modification apportée à un lectuer peut prendre jusqu'à 5 minutes pour apparaître dans les lectuers publiés.

Écran de fin social

Quand le Social est sélectionnée, la boîte de dialogue de partage social s'affiche à la fin de la vidéo.

Le partage social pour le lectuer doit être activé pour utiliser l'écran de fin du partage social. Si la Social est sélectionnée, le partage social sera activé automatiquement. Les options de partage sur les réseaux sociaux qui s'affichent reflètent ce qui a été configuré dans les paramètres sociaux. Pour plus d'informations sur les paramètres sociaux, consultez Configurer le social Propriétés pour un lectuer.

Écran de fin personnalisé

Lorsque l'option Custom est sélectionnée, du HTML personnalisé peut être utilisé pour afficher du texte ou des images supplémentaires au-dessus des icônes de relecture et de partage. Pour configurer un écran de fin personnalisé, sélectionnez Douane et collez-le en HTML à afficher dans le HTML personnalisé propriété.

Note: Une vidéo peut être utilisée en ajoutant l'élément <video> html à un code HTML personnalisé.
Note: Le partage doit être activé pour que l'icône de partage apparaisse sur l'écran de fin.
Note: Si la Personnalisé est sélectionnée et qu'aucun code HTML n'est fourni, le bouton de relecture s'affichera à la fin de la vidéo.
