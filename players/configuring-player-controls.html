Les contrôles du lectuer tels qu'un sélecteur de qualité, le taux de lecture, le contrôle plein écran et le curseur de volume vertical peuvent être configurés pour un lectuer.

Si vous voyez un contrôle qui apparaît comme illustré ici :

il active la fonctionnalité d'image dans l'image. Voir le Image dans l'image plus loin dans ce document pour des informations complètes.

Pour configurer le Les contrôles propriétés, procédez comme suit:

  1. Cliquez sur Players dans l'en-tête de navigation pour ouvrir le module Players.
  2. Cliquez sur le lien du lecteur à configurer.
  3. Cliquez sur Contrôles dans le menu de navigation de gauche.

Les contrôles suivants peuvent être configurés :

  • Sélecteur de qualité
  • Taux de lecture
  • Plein écran
  • Curseur de volume vertical
  • Recherche de vignettes
  • Passer à la vidéo suivante (playlist)

Une fois les modifications apportées, assurez-vous de republier le lecteur pour que les modifications soient visibles.

Noter: Toute modification apportée à un lectuer peut prendre jusqu'à 5 minutes pour apparaître dans les lectuers publiés.

Sélecteur de qualité

Activation du Sélecteur de qualité offre la possibilité de sélectionner manuellement un rendu différent pendant la lecture de la vidéo. Une icône d'engrenage apparaîtra dans le coin inférieur droit du lectuer.

Note: Le sélecteur de qualité nécessite Brightcove Player version 5.17.0 ou plus récente et peut ne pas fonctionner avec tous les navigateurs.

Lorsque le sélecteur de qualité est activé, deux options pour l'étiquette du sélecteur de qualité sont activées:

  • Afficher la résolution (par exemple, 720 p) - Les rendus seront affichés par des lignes de résolution horizontale

    Lorsque vous utilisez cette option, un autre paramètre (resolutionLabelBitrates) peut être ajouté au plugin Quality Selection pour afficher également les débits binaires.

    Cela doit être fait en éditant les paramètres du plugin car il n'y a actuellement aucun moyen de configurer cette option à l'aide de l'interface utilisateur du module Players. Pour plus d'informations, consultez le Plugin de sélection de qualité document.

  • Afficher le HD/SD - Le lecteur examinera les informations de débit de chaque rendu pour déterminer s'il s'agit d'une carte SD ou HD

Taux de lecture

Activation Taux de lecture offre la possibilité de sélectionner manuellement une vitesse de lecture différente. Une icône de taux de lecture apparaît dans le coin inférieur droit du lecteur.

Remarques:
  • Le sélecteur de taux de lecture nécessite Brightcove Player version 6.35.0 ou ultérieure et peut ne pas fonctionner avec tous les navigateurs.
  • La largeur du lecteur doit être d'au moins 424 pixels pour afficher le contrôle des taux de lecture.

Plein écran

Sélectionner Plein écran pour activer le contrôle plein écran. Le contrôle plein écran permet de redimensionner le lecteur pour remplir tout l'écran. Au lancement de la vidéo, une icône de plein écran s'affiche en bas à droite du lecteur.

Curseur de volume vertical

Sélectionner Curseur de volume vertical pour afficher le curseur de volume verticalement.

Cette fonctionnalité n'est disponible que dans Brightcove Player 6.32.0+.

Recherche de vignettes

Sélectionner Recherche de vignettes pour afficher des miniatures vidéo sur la barre de progression.

Cette fonctionnalité n'est disponible que dans Brightcove Player 6.41.0+. Pour en savoir plus sur la recherche de vignettes, voir Ajout d'une recherche de vignettes à un lecteur Brightcove.

Passer à la vidéo suivante (playlist)

Sélectionner Passer à la vidéo suivante (playlist) pour activer le contrôle de saut. Vous pouvez cliquer sur le contrôle de saut pour passer à la vidéo suivante dans la liste de lecture.

Image dans l'image

HTML5 image dans la fonctionnalité d'image est définie comme suit: Permet aux sites Web de créer une fenêtre vidéo flottante toujours au-dessus des autres fenêtres afin que les utilisateurs puissent continuer à consommer des médias tout en interagissant avec d'autres sites de contenu ou applications sur leur appareil.

Un exemple d'image dans l'image est illustré dans les deux captures d'écran suivantes. Cette première capture d'écran montre un navigateur avec un lecteur affichant une vidéo. Notez le bouton image dans l'image en bas à droite du lectuer avec la flèche rouge pointant vers lui.

Cette capture d'écran suivante montre le résultat du clic sur le bouton Image dans l'image. Le lectuer se déplace par défaut près du coin inférieur droit de l'écran (PAS du navigateur). Le lectuer peut alors être glissé partout où le spectateur souhaite le placer.

Cette fonctionnalité est activée par défaut. Tous les navigateurs ne prennent pas en charge cette fonctionnalité image dans l'image. Le tableau suivant détaille la prise en charge actuelle.

Navigateur Support technique
Chrome Prise en charge dans la version 78+
Safari Prise en charge dans la version 13.1+
Firefox Prise en charge dans la version 13.1+
Bord Prise en charge dans la version 78+
