Les playlist-endscreen Le plugin permet d'afficher une liste de lecture de vidéos sur l'écran de fin du lecteur. Le spectateur peut alors cliquer sur une image miniature de vidéo pour regarder une autre vidéo. Dans ce lecteur, un écran de fin de liste de lecture a été configuré.

Pour plus d'informations sur les autres options des écrans de fin de lecteur, consultez Configuration de l'écran de fin du lecteur.

Note: Pour plus d'informations sur la configuration d'un écran de fin de liste de lecture par programme, consultez la documentation des développeurs.

Configuration du plugin playlist-endscreen

Pour implémenter le plug-in playlist-endscreen à l'aide du module Players, procédez comme suit :

  1. Ouvrez le module Players et trouvez le lecteur.
  2. Cliquez sur le lien du lecteur pour ouvrir les propriétés de celui-ci.
  3. Cliquez sur Plugins dans la navigation de gauche.
  4. Cliquez sur Ajouter un plugin> Plugin Brightcove.
  5. Pour le Plugin Brightcove , sélectionnez playlist-endscreen.
  6. Les Options (JSON) contient le
    • ID de la liste de lecture - ID de la liste de lecture à afficher sur l'écran de fin
    • Titre (facultatif) - Titre à afficher au dessus de la liste des vidéos
    7. {
  "playlistId": "1234567001",
  "title": "Related Videos"
}
  7. Cliquez sur Enregistrer, puis sur Publier le lecteur. Notez que cela peut prendre jusqu'à 5 minutes pour que les modifications apparaissent dans les lecteurs publiés.
