Les lecteurs peuvent être configurés de façon à ce que le contenu vidéo soit partagé sur les réseaux sociaux les plus populaires. Lorsque le partage sur les réseaux sociaux est activé, une icône de partage s'affiche dans le dock de titre et de description.

Note: Si les paramètres du lecteur sont définis pour ne pas afficher le titre et la description, l'icône de partage apparaît en bas du lecteur à côté de la tête de lecture.

Lorsque l'utilisateur clique sur l'icône de partage ; les options de partage sélectionnées s'affichent.

Pour configurer le Social propriétés, procédez comme suit :

  1. Cliquez sur Players dans l'en-tête de navigation pour ouvrir le module Players.
  2. Cliquez sur le lien du lecteur à configurer.
  3. Cliquez sur Social dans le menu de navigation de gauche.
  4. Cochez la case pour Activer le partage et sélectionnez les sites de médias sociaux sur lesquels la vidéo peut être partagée.
  5. Vérifier Afficher le code intégré du lecteur pour permettre au spectateur de copier le code d'intégration du lecteur lorsque les options de partage social sont affichées.
  6. Vérifier Afficher le lien direct du lecteur pour permettre au spectateur de copier un lien vers la page où le lecteur est intégré lorsque les options de partage social sont affichées.
