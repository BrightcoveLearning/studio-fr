Pour configurer ou modifier les paramètres du lecteur, ouvrez le module Lecteurs et cliquez sur un nom de lectuer. Pour modifier les propriétés du lecteur, cliquez sur un élément dans le menu de navigation de gauche.

Les propriétés suivantes peuvent être configurées :

  • Informations sur le lectuer
  • Coiffant
  • Relecture
  • Contrôles
  • Social
  • Plugins
  • Écrans de fin
  • La publicité
  • Restriction de contenu
  • Campagne
  • Éditeur JSON
  • Analyse tierce
Note: Toute modification apportée à un lectuer peut prendre jusqu'à 5 minutes pour apparaître dans les lectuers publiés.

Player Information

Les Informations sur le lectuer Les propriétés sont utilisées pour configurer le nom du lecteur, la taille, le rapport hauteur/largeur, le mode de mise à jour des dimensions et la version. Pour plus d'informations, consultez Définition des propriétés des informations sur le lecteur.

Styling

Les Coiffant Les propriétés sont utilisées pour configurer le type de lecteur, l'apparence du lecteur, les couleurs et une superposition de lecteur. Pour plus d'informations, consultez la rubrique Personnalisation du style des lecteurs.

Lecture

Les Relecture Les propriétés sont utilisées pour configurer les options de lecture vidéo. Pour plus d'informations, consultez Configuration des propriétés de lecture du lecteur.

Commandes

Les Les contrôles Les propriétés sont utilisées pour configurer les contrôles qui apparaissent dans la barre de contrôle du lecteur. Pour plus d'informations, consultez Configuration des commandes du lecteur.

Réseaux sociaux

Les Social Les propriétés sont utilisées pour configurer les options sociales pour le lectuer. Pour plus d'informations, consultez Configuration des propriétés sociales d'un lectuer.

Plug-ins

Plugins utilisez une combinaison de HTML, JavaScript et / ou CSS pour personnaliser le comportement du lecteur. Ils peuvent être développés pour :

  • Modifier le comportement par défaut d'un lectuer
  • Ajout de fonctionnalités à un lecteur
  • Personnalisation de l'apparence du

Pour plus d'informations, consultez la rubrique Configuration des plug-ins du lecteur.

Écrans de fin

Les Écrans de fin Les propriétés sont utilisées pour configurer ce qui est affiché lorsqu'une vidéo se termine. Pour plus d'informations, consultez l'article Configuration de l'écran de fin du lecteur.

Advertising

Les Publicité Les propriétés sont utilisées pour activer les annonces IMA dans un lecteur. Pour plus d'informations, consultez la rubrique Configuration des publicités du lecteur.

Restriction du contenu

Les Restriction de contenu Les propriétés sont utilisées pour configurer les restrictions de domaine et le filtrage géographique pour le lecteur. Pour plus d'informations, consultez Configuration des restrictions de contenu du lecteur.

Campagne

Les Campagne Les propriétés sont utilisées pour intégrer les Brightcove Players aux plateformes d'automatisation du marketing et aux formulaires pour prospects. Pour plus d'informations, consultez Création d'un lecteur compatible avec la campagne.

Éditeur JSON

L'éditeur JSON peut être utilisé pour modifier la configuration du lecteur. Des précautions doivent être prises lors de la modification de la configuration du lecteur.

Note: Il s'agit d'une option avancée qui ne doit être utilisée que par les éditeurs qui sont à l'aise de modifier la configuration du lecteur.

Pour plus d'informations sur la modification de la configuration du lecteur, consultez le Guide de configuration du lecteur.

Systèmes de statistiques tiers

Les Analyses tierces Les propriétés sont utilisées pour intégrer les Brightcove Players à Google Analytics et Adobe Analytics. Pour plus d'informations, consultez Configuration du suivi des événements Google Analytics et Configuration du suivi des événements Adobe Analytics.
