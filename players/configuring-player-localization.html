Les lecteurs Brightcove peuvent être localisés de sorte que les éléments d'interface apparaissent dans différentes langues. Si les propriétés de localisation du lecteur ne sont pas configurées, l'interface s'affiche en anglais.

Pour configurer la localisation du lecteur, procédez comme suit.

  1. Ouvrez le module Players.
  2. Cliquez sur un lecteur pour accéder aux propriétés.
  3. Cliquez sur Informations sur le lecteur dans le menu de navigation de gauche.
  4. Dans le Localisation champ, entrez le code de langue à deux lettres pour la langue et appuyez sur Entrée. Au fur et à mesure que vous tapez, la liste des langues sera filtrée. Vous pouvez également cliquer sur une valeur pour la sélectionner.
    Note: La liste des langues prises en charge se trouve sur le Langage Video.js page.
  5. Ajoutez des langues supplémentaires, le cas échéant. Si un code de langue non valide est entré, il apparaît en rouge.
  6. Cliquez sur Save.

Assurez-vous de publier les modifications apportées au lecteur.

Vérification de la langue

Pour utiliser la langue localisée, définissez la langue préférée dans le navigateur. Les étapes exactes varient en fonction du navigateur et du système d'exploitation. Si la langue préférée du navigateur est l'anglais, ou si une langue n'est pas disponible, vous verrez le texte par défaut, ou en anglais,. Les navigateurs n'ont pas de détection des paramètres régionaux parfaitement standardisés, de sorte que cela peut être un peu compliqué. Globalement, les navigateurs cherchent à déterminer la langue souhaitée dans cet ordre :

  • Vérifiez si la page d'intégration a un attribut lang sur le <html> élément
  • Vérifier les configurations de paramètres régionaux spécifiques au navigateur
  • Retour à l'anglais
Note: Cette technique ne fonctionnera qu'avec l'implémentation de code avancé (dans la page). Étant donné que l'implémentation Standard se charge dans un iframe, le paramètre de langue du navigateur ne s'y trouve pas.

Si vous utilisez l'attribut HTML, vous n'avez pas besoin de définir la langue par défaut dans votre navigateur ou votre ordinateur car le lecteur utilisera celle du lang attribut.

<html lang="es">

De plus, si vous utilisez un iframe, gardez à l'esprit que vous devrez ajouter le paramètre de requête à l'URL du lecteur.

//players.brightCove.net/6045682556001/JR6QQMDSB_default/index.html ? videoID=6054419564001&language=ES
