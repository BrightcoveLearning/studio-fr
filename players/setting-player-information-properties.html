Les propriétés des informations du lecteur peuvent être utilisées pour définir le nom du lectuer, la taille, le rapport d'aspect, les dimensions, le mode de mise à jour et la version du lecteur.

De nombreuses propriétés du lecteur peuvent également être réglées/modifiées dynamiquement en modifiant le code de publication du lecteur. Pour plus d'informations, consultez ces documents associés selon que vous utilisez ou non le Standard (iframe) ou Avancé (encart) code intégré.

Pour configurer le Informations sur le lectuer propriétés, procédez comme suit :

  1. Cliquez sur Players dans l'en-tête de navigation pour ouvrir le module Players.
  2. Cliquez sur le lien du lecteur à configurer.
  3. Cliquez sur Informations sur le lecteur dans le menu de navigation de gauche.
Note: Toute modification apportée à un lectuer peut prendre jusqu'à 5 minutes pour apparaître dans les lectuers publiés.

Les propriétés d'informations de lecteur suivantes peuvent être configurées :

  • Nom et description abrégée
  • Type de lectuer
  • Dimensionnement
  • Ratio d'aspect
  • Dimensions du lectuer
  • Mode de mise à jour
  • Version du lectuer
  • Localisation

Après avoir apporté des modifications, cliquez sur sauver puis publiez le lecteur.

Type de lectuer

Les Type de lectuer Ce paramètre détermine si le lecteur affichera une seule vidéo ou si les listes de lecture sont prises en charge :

  • Vidéo unique: une seule vidéo est affichée
  • Playlist - Le lecteur prend en charge les listes de lecture

Sélection Playlist activera des Coiffant et Relecture Propriétés.

Redimensionnement

Le paramètre Sizing peut être défini sur Responsive ou Fixed.

  • Responsive: la taille du lecteur se réglera automatiquement en fonction de la taille du navigateur.
  • Fixed: la taille du lecteur est définie en fonction des paramètres de largeur et de hauteur.
Note: Si vous sélectionnez le Sensible option de dimensionnement, le code Standard (iframe) fonctionne bien et le lecteur et la liste de lecture sont réactifs. Si vous utilisez le code Advanced (intégré dans la page), le lecteur sera réactif, mais pas la liste de lecture.

Rapport des dimensions

Le rapport d' aspect définit le rapport hauteur/largeur du lectuer.

  • 1:1, 3:2, 4:3, 16:9, 21:9 ou 9:16 - Entrez la largeur d'un lectuer et la hauteur sera déterminée automatiquement
  • Personnalisé - Saisissez des valeurs pour la largeur et la hauteur du lecteur

Dimensions du lecteur

Le paramètre Player Dimensions contrôle la hauteur et la largeur du lecteur.

Update Mode and Player Version

Le paramètre Update Mode contrôle la gestion des mises à jour du lecteur. La version du lecteur permet de sélectionner une version de lecteur. Les Version du lecteur publiée champ affichera la version actuelle du lecteur. Pour plus d'informations sur ces paramètres et leur utilisation correcte, consultez la rubrique Paramétrage du mode de mise à jour du lecteur et de sa version.

Localisation

Les Localisation peut être utilisé pour inclure la ou les langues que vous souhaitez utiliser pour la localisation du lecteur. Pour plus d'informations sur la localisation des lectuers, consultez Configuration de la localisation du lecteur.
