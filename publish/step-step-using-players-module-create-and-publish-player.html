Après avoir suivi les étapes de cette rubrique, vous devriez être en mesure de :

  • Naviguez à l'intérieur du module Players
  • Créer un lectuer
  • Définir les propriétés du lecteur
  • Transmettre un identifiant vidéo Video Cloud à afficher dans le lecteur
  • Prévisualisez et publiez un lecteur
Note: Si vous souhaitez publier une playlist ou une vidéo unique sur un lecteur, utilisez la fonction de publication dans le module Media. Pour plus d'informations, voir Publication de vidéos et de listes de lecture avec le module Media.

Audience

Tout éditeur vidéo qui doit créer et publier des vidéos avec le module Players.

Conditions préalables

  • Un compte Video Cloud avec accès au module Players.
  • Une vidéo dans votre compte Video Cloud pour la publication. La vidéo utilisée dans cette brève introduction est téléchargeable ici ou vous pouvez utiliser votre propre contenu.
  • Si vous utilisez la vidéo fournie dans cette brève introduction, chargez-la sur votre compte Video Cloud.

Instructions

Pour créer et publier un lecteur Brightcove avec Video Cloud Studio, vous devrez effectuer les tâches suivantes :

  • Créer un lectuer
  • Configurer les dimensions du lecteur
  • Publier et prévisualiser le lecteur
  • Attribuer dynamiquement du contenu vidéo au lecteur
  • Configurer les propriétés de style
  • Configurer les propriétés de partage social
  • Aperçu du lecteur
  • Publier le lectuer final

Créer un lectuer

Pour créer un lecteur, procédez comme suit.

  1. Connectez-vous à Video Cloud Studio.
  2. Le nom du compte Video Cloud actuel s'affiche dans le coin supérieur droit de la page. Si vous avez plusieurs comptes, cliquez sur le utilisateur actuel - compte actuel lien et sélectionnez le compte dans lequel vous souhaitez créer le lectuer.
  3. Cliquez sur le lien Players dans l'en-tête de navigation.
  4. Cliquez sur le bouton New Player.
  5. Remplissez les champs Name et Short Description du lecteur. Dans le cadre de ce démarrage rapide, le nom du lecteur est Sample Player.
  6. Vérifiez que le lecteur figure sur la liste des lecteurs.

Quand un lecteur est créé avec le module Players, il est conçu avec la dernière version publiée par Brightcove. Pour afficher la version du lecteur, ouvrez le module Players et cliquez sur le lecteur à vérifier. Les propriétés Informations sur le lectuer afficheront le Version du lecteur publiée.

Tous les lecteurs peuvent ne pas avoir la même version. Brightcove publiera de nouvelles versions du lecteur Brightcove à mesure du développement et de l'amélioration de celui-ci. Par défaut, les mises à jour sont appliquées automatiquement à tous les lecteurs. Pour empêcher la mise à jour automatique d'un lecteur, vous devez définir le paramètre Update Mode sur Manual (aussi connu sous le nom de verrouillage de lecteur). Pour plus d'informations, consultez la rubrique Paramétrage du mode de mise à jour du lecteur et de sa version.

Paramétrer les dimensions du lecteur

En ajoutant les dimensions du lecteur, les paramètres width et height seront intégrés au code d'intégration iframe. Si vous laissez les dimensions vides, le lecteur peut être mis dans une <div> de n'importe quelle taille, qui s'étendra pour couvrir tout l'espace disponible sur la page. Pour configurer les dimensions du lecteur, procédez comme suit.

  1. Cliquez sur le lien de Sample Player pour ouvrir la page des propriétés du lecteur.
  2. Cliquez sur Informations sur le lecteur dans le menu de navigation de gauche.
  3. Met le Ratio d'aspect à 16:9.
  4. Met le Largeur du lectuer à 640.
  5. Cliquez sur Save.

Pour plus d'informations sur les propriétés des informations sur le lecteur, consultez Définition des propriétés des informations sur le lecteur.

Publier et prévisualiser le lecteur

Publier le lecteur propage les modifications apportées au lecteur publié. Pour plus d'informations sur les lecteurs d'aperçu, les lecteurs publiés et le code de publication des lecteurs, consultez Génération de Player Embed Code.

Pour publier le lecteur, procédez comme suit.

  1. Cliquez sur Publier et intégrer et alors Publier les modifications pour publier le lecteur.
  2. Vérifiez que la publication s'est bien déroulée.
  3. Cliquez sur l'URL du lecteur pour ouvrir le lecteur d'aperçu dans un nouvel onglet de navigateur. Le lecteur sera vide car aucune vidéo n'a été attribuée au lecteur.

Affecter le contenu vidéo au lecteur de façon dynamique

Pour cette brève introduction, nous attribuerons de façon dynamique un ID de vidéo Video Cloud au lecteur en utilisant un paramètre URL. Pour ce faire, procédez comme suit :

  1. Dans le module Médias, cherchez la vidéo à publier. L'ID vidéo apparaîtra sous le nom de la vidéo.
  2. Copiez cet ID dans le presse-papiers.
  3. Revenez à l'onglet avec le lecteur de prévisualisation. Pour transmettre dynamiquement l'ID vidéo à afficher, ajoutez ce qui suit à l'URL et appuyez sur Entrée :
    ?videoId=your video id
    Noter: Le nom du paramètre URL est sensible à la casse.

    L'URL complète ressemblera à quelque chose comme :

    https://players.brightcove.net/1486906377/Ctzra7vEL_default/index.html?videoId=5798846643001
  4. Vérifiez que la vidéo s'affiche.

Configuration des propriétés de style

Les propriétés de style permettent de personnaliser l'apparence du lecteur. Pour plus d'informations sur la modification du style des lecteurs, consultez la rubrique Configurer des paramètres de style du lecteur.

Pour configurer les paramètres de style, revenez dans le module Players et procédez ainsi.

  1. Cliquez sur le lien de Sample Player pour ouvrir la page des propriétés du lecteur.
  2. Cliquez sur Style dans le menu de navigation de gauche.
  3. Sélectionner un Forme du bouton de lecture et Position du bouton de lecture.
  4. Dans le Thème de couleur section, cliquez sur le Bouton jouer pipette à couleurs.
  5. Cliquez sur la couleur rouge sur le côté droit. Cliquez sur le carré pour sélectionner une nuance de rouge.
  6. Cliquez en dehors de la palette pour la fermer.
  7. Notez que sous le lecteur de prévisualisation, il y a un message indiquant que des modifications ont été apportées au lecteur.

Configurer les propriétés de partage sur les réseaux sociaux

Lorsque le partage sur les réseaux sociaux est activé, une icône de partage s'affiche en bas du lecteur, à côté de la barre de progression. Cela permet aux utilisateurs de partager votre contenu sur les réseaux sociaux les plus populaires. Pour plus d'informations sur la configuration des propriétés sociales d'un lectuer, consultez Configuration des propriétés sociales d'un lectuer.

Pour configurer les paramètres sociaux, procédez ainsi.

  1. Cliquez sur Social dans le menu de navigation de gauche.
  2. Vérifier la Activer le partage case à cocher.
  3. Sélectionnez tous les réseaux sociaux.

Prévisualiser le lecteur

Le lecteur de prévisualisation sur la page doit refléter les changements de style et de réseaux sociaux qui ont été apportés au lectuer.

Note: Si vous revenez sur le lecteur publié que nous avons ouvert plus tôt et que vous actualisez la page, vous remarquerez que ces modifications ne sont pas actives dans le lecteur publié. Les changements de couleur et de réseaux sociaux ne seront pas transférés vers le lecteur publié tant que les modifications n'auront pas été publiées dans le module Players.

Il est également possible de créer le code de publication du lecteur de prévisualisation. Cela peut être utile, par exemple, si le service marketing doit valider les modifications apportées au lecteur. Vous pouvez alors leur envoyer une URL vers le lecteur de prévisualisation. Pour créer le code de publication du lecteur de prévisualisation, procédez comme suit.

  1. Sur la page d'accueil du module lectuers, cliquez sur Code intégré et URL et cliquez Aperçu du lecteur. La fenêtre Preview Player Embed Code s'affiche.
  2. Cliquez sur l'URL du lecteur pour ouvrir le lecteur d'aperçu dans un nouvel onglet de navigateur.
  3. Le lecteur de prévisualisation doit occuper la totalité de la longueur de la page du navigateur et posséder un filigrane Brightcove. Vérifiez que les modifications apportées à la promotion virale et aux couleurs sont visibles.
  4. Laissez le lecteur ouvert et revenez au module Players.

Publication du lecteur terminé

Les modifications au lecteur de prévisualisation ne seront pas visibles dans le lecteur publié tant qu'elles n'auront pas été elles-mêmes publiées. Pour publier les modifications, procédez comme suit.

  1. Dans le module lectuers, cliquez sur Publier et intégrer et alors Publier les modifications pour publier le lecteur.
  2. Vérifiez que la publication s'est bien déroulée.
  3. Revenez à l'onglet du navigateur affichant le lecteur publié. Souvenez-vous que le lecteur de prévisualisation doit afficher un filigrane Brightcove.
  4. Actualisez la page et vérifiez que les modifications de couleur et les paramètres de promotion virale sont bien affichés dans le lecteur publié. Il peut y avoir un léger délai avant la prise en compte des modifications dans le lecteur publié.
