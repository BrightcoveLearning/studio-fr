{{ page.description }}

Vous pouvez avoir des cas où vous devez changer la vidéo associée à un lecteur. Par exemple, le spectateur a peut-être la possibilité de contrôler la vidéo qu'il souhaite voir. Peut-être que la vidéo dans le lecteur est basée sur la partie du site d'où le spectateur provient.

Si vous avez besoin d'afficher différentes vidéos dans un lecteur, il est recommandé d'utiliser un lecteur, puis de modifier dynamiquement la vidéo associée au lecteur.

Les techniques suivantes sont décrites dans cette rubrique :

  • Passer l'ID vidéo en tant que paramètre d'URL
  • Passer l'ID de référence en tant que paramètre d'URL
  • Modification de l'intégration code pour fournir un ID vidéo
  • Modification de l'intégration code pour fournir un ID de référence
Note: Bien qu'il soit possible de modifier une vidéo liée statiquement à un lecteur, ce n'est pas une bonne pratique. Si vous procédez ainsi, l'image d'affiche de la vidéo liée statiquement peut apparaître brièvement dans le lecteur avant que la vidéo ajoutée dynamiquement ne s'affiche. Pour cette raison, il est recommandé de créer un lecteur auquel aucune vidéo n'est initialement associée pour être utilisée dans des situations où la vidéo sera modifiée dynamiquement.

Le contenu peut également être attribué dynamiquement à un lecteur à l'aide de l'API de catalogue. Pour plus d'informations et un exemple de code, consultez le document du développeur Changer dynamiquement les vidéos sources.

Passer l'ID vidéo en tant que paramètre d'URL

Vous pouvez modifier la source vidéo d'un lecteur en transmettant l'ID vidéo en tant que paramètre d'URL à l'aide du formulaire ?videoId=YOUR_VIDEO_ID.

http://players.brightcove.net/20318290001/f1d70e07-8480-4bbd-b4ef-747e9333a034_default/index.html?videoId=4292158104001

Passer l'ID de référence en tant que paramètre d'URL

Vous pouvez modifier la source vidéo d'un lecteur en transmettant l'ID de référence en tant que paramètre d'URL à l'aide du formulaire ?videoId=ref:YOUR_VIDEO_REF_ID.

http://players.brightcove.net/20318290001/f1d70e07-8480-4bbd-b4ef-747e9333a034_default/index.html?videoId=ref:1234

Modification du code d'intégration pour fournir un ID vidéo

Vous pouvez attribuer une vidéo Video Cloud au code d'intégration du lecteur directement à l'aide de l'ID vidéo.

Code d'intégration dans la page

Ajoutez l'attribut data-video-id="YOUR_VIDEO_ID" à la <video> balise :

<video-js   data-account="57838016001"
  data-player="6e44f16e-31e9-48b2-83b9-6edd71217540"
  data-embed="default"
  data-video-id="68143720001"
  class="video-js"
  controls></video-js>
<script src="https://players.brightcove.net/57838016001/6e44f16e-31e9-48b2-83b9-6edd71217540_default/index.min.js"></script>
Note: Lorsque le JavaScript du lecteur s'exécute, il lit l'ID vidéo à partir de l'attribut data-video-id uniquement au démarrage. La modification de la valeur avec, par exemple, la méthode setAttribute () de JavaScript, n'entraîne pas le chargement de la vidéo par le lecteur. Pour un exemple sur la façon de modifier la vidéo à l'aide de JavaScript, consultez le Catalogue de lectuers document de développeur.

iframe Code d'intégration

Ajoutez ?videoId=YOUR_VIDEO_ID à la fin de l'URL dans l' src attribut de l'iframe :

<iframe src='//players.brightcove.net/57838016001/6e44f16e-31e9-48b2-83b9-6edd71217540_default/index.html?videoId=68143720001'
 allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>

Modification du code d'intégration pour fournir un ID de référence

Vous pouvez attribuer une vidéo Video Cloud au code d'intégration du lecteur directement à l'aide de l'ID de référence.

Note: Lorsque vous utilisez un ID de référence contenant des caractères spéciaux, vous devez échapper aux caractères spéciaux du code de publication, sinon des problèmes peuvent survenir.

Code d'intégration dans la page

Ajoutez l'attribut data-video-id="ref:YOUR_VIDEO_REF_ID" à la <video> balise :

<video-js data-account="57838016001"
  data-player="6e44f16e-31e9-48b2-83b9-6edd71217540"
  data-embed="default"
  data-video-id="ref:68143720001"
  class="video-js"
  controls></video-js>
<script src="https://players.brightcove.net/57838016001/6e44f16e-31e9-48b2-83b9-6edd71217540_default/index.min.js"></script>

iframe Code d'intégration

Ajoutez ?videoId=ref:YOUR_VIDEO_REF_ID à la fin de l'URL dans l' src attribut de l'iframe :

<iframe src='//players.brightcove.net/57838016001/6e44f16e-31e9-48b2-83b9-6edd71217540_default/index.html?videoId=ref:68143720001'
 allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>
