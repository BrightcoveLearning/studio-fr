Les articles instantanés Facebook sont une expérience rapide et interactive pour lire des articles dans un fil d'actualité Facebook. Les articles instantanés peuvent être publiés directement à partir d'un système de gestion de contenu (à l'aide de l'API Instant Articles) ou en fournissant un flux RSS. Les articles instantanés s'affichent jusqu'à dix fois plus rapidement que les articles Web mobiles standard. Lorsqu'un ami ou une page que vous suivez partage un lien dans votre fil d'actualité, Facebook vérifiera s'il existe un article instantané associé à cette URL. Si tel est le cas, Facebook l'affichera sous forme d'article instantané. Sinon, il s'ouvrira dans le navigateur Web.

Les lecteurs Brightcove intégrés à Instant Article continuent de prendre en charge :

  • Publicité pré-roll et mid-roll
  • Analyse de Video Cloud
  • Plugins tiers

Notez que tandis que les analyses seront collectées dans Video Cloud, le domaine d'affichage enregistré dans Analytics sera un brightcove.com domaine, pas un facebook.com domaine.

Bien que différents types de contenu puissent être partagés à l'aide d'articles instantanés, cette rubrique couvre le partage des lecteurs Brightcove dans les pages d'articles instantanés. Pour obtenir des informations complètes sur les articles instantanés, consultez le Documentation Facebook.

Génération du code d'intégration du lecteur

Aucune modification des propriétés du lecteur n'est nécessaire pour l'utiliser dans un article instantané. Suivez ces étapes pour générer le code d'intégration du lecteur pour un article instantané :

  1. Prenez le code d'intégration iframe standard de Video Cloud Studio et placez-le dans un HTML5 <figure> élément. 
    	<chiffre>< iframe src="//players.brightcove.net/1486906377/V1ziujO35l_default/index.html?videoId=3766858011001" allowfullscreen webkitallowfullscreen mozallowfullscreen width="640" height="360">< /iframe>< /chiffre>
    En utilisant l'identifiant de référence vidéo (c'est-à-dire videoId="ref:referenceId" ) dans le code d'intégration iframe est pris en charge lors de la publication sur Facebook Instant Articles.
  2. Ajouter le op-interactive classe à la figure marque. 
    	<chiffre class="op-interactif">< iframe src="//players.brightcove.net/1486906377/V1ziujO35l_default/index.html?videoId=3766858011001" allowfullscreen webkitallowfullscreen mozallowfullscreen width="640" height="360">< /iframe>< /chiffre>
  3. Donnez à l'iframe un width et height. Le width et height définit le rapport hauteur/largeur plutôt que la taille absolue du lectuer. Le code fini ressemblera à ceci : 
    	<figure class="op-interactive">< iframe src="//players.brightcove.net/1486906377/V1ziujO35l_default/index.html?videoId=3766858011001" allowfullscreen webkitallowfullscreen mozallowfullscreen largeur="16" hauteur="9">< /iframe>< /chiffre>
