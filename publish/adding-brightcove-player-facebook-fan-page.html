L'exemple ci-dessous montre un lecteur intégré sur une page de fan Facebook.

Installation de l'application HTML statique

Le processus décrit ci-dessous utilise une application que vous ajoutez à votre compte Facebook. Cette application crée un onglet sur votre page Facebook, dans lequel vous pouvez coller le HTML statique de ce que vous voulez y afficher. Plusieurs autres applications proposent une telle fonctionnalité. Pour effectuer les étapes ci-dessous, vous devrez avoir créé une page Facebook. Si ce n'est pas le cas, cliquez ici pour en créer une.

  1. Ajoutez l'application « HTML statique: onglets iFrame » en tant qu'onglet sur votre page de fan. Pour ajouter cette application, accédez à https://apps.facebook.com/static_html_plus. Suivez les instructions pour installer l'application sur votre compte.
  2. Une fois l'application installée, vous devriez voir un onglet Welcome (Bienvenue) sur votre page de fan. Clique le Bienvenue languette.
  3. Cliquez sur Onglet Configurer. Cela ouvrira un éditeur où vous pouvez coller en HTML.
  4. Dans Video Cloud Studio, copiez le code d'intégration Standard (iframe) du lecteur que vous souhaitez ajouter à votre page Facebook Fan.
  5. Collez le code d'intégration dans l'éditeur HTML statique.
  6. Cliquez sur Aperçu pour vérifier que votre lecteur s'affiche correctement.
  7. Si l'aperçu était correct, cliquez sur Enregistrer et publier pour enregistrer vos modifications.
  8. Pour afficher le lecteur, ouvrez votre page Facebook et cliquez sur le bouton Bienvenue languette.

Modification du nom de l'onglet

Vous pouvez changer le nom du Bienvenue onglet à quelque chose de plus descriptif en suivant les étapes ci-dessous.

  1. Connectez-vous à votre compte Facebook.
  2. Ouvrez votre page Facebook où vous avez ajouté le lectuer.
  3. Cliquez sur le lien Paramètres en haut de la page.
  4. Cliquez sur Modifier la page dans la navigation de gauche.
  5. Localisez le Bienvenue onglet et cliquez Réglages.
  6. Cliquez sur Modifier les paramètres.
  7. Une fenêtre s'affiche et vous permet de renommer l'onglet. Vous avez également la possibilité de le supprimer et de modifier l'icône qui lui est associée. Entrez un nouveau nom pour l'onglet.
  8. Cliquez sur sauver et alors D'accord.
  9. Cliquez sur Save. Notez que l'interface utilisateur des paramètres Facebook affichera toujours un nom d'onglet Bienvenue mais le nouveau nom apparaîtra lorsque vous afficherez la page.

Modification du code HTML

Si vous devez modifier le code HTML associé à l'onglet, suivez ces étapes.

  1. Connectez-vous à votre compte Facebook et accédez à votre page.
  2. Cliquez sur l'onglet qui représente votre page vidéo, puis cliquez sur Onglet Modifier.
  3. Cliquez sur le code HTML approprié. Cliquez sur Enregistrer et publier lorsque vous avez terminé.
