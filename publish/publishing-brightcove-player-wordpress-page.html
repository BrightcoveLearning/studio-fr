Brightcove fournit également le Plugin WordPress Brightcove Video Connect. Ce plugin prend en charge plusieurs comptes Video Cloud et vous permet de télécharger des vidéos, de les ajouter à des listes de lecture et de rendre des shortcodes avec vos vidéos le tout depuis l'interface d'administration WordPress.

Étapes à suivre pour publier un lecteur Brightcove dans une page WordPress

Les étapes et les captures d'écran ont été effectuées à l'aide de WordPress 5.2.2.

  1. Connectez-vous à votre site WordPress et accédez à Des postes sur la barre latérale gauche.
  2. Cliquez sur Ajouter nouveau pour ajouter un nouveau message.
  3. Ajoutez un titre pour la publication.
  4. Clique le + pour ajouter un bloc. Dans le Mise en page rubrique, cliquez sur HTML personnalisé pour ajouter un bloc HTML.
  5. Collez le code d'intégration iframe dans l'éditeur. Modifier le width et height paramètres selon les besoins.
  6. Cliquez sur Aperçu pour prévisualiser le lecteur.
  7. Cliquez sur Publier... Vérifiez les paramètres de publication, puis cliquez sur Publier.
  8. Pour afficher le lecteur sur votre site, cliquez sur le bouton Voir l'article bouton sur le côté droit de la page.
  9. Si vous modifiez le code ou le titre, cliquez sur le bouton Update (Mettre à jour) pour enregistrer vos modifications.
