Google Sites est un outil de création de pages web proposé par Google. Les lecteurs Brightcove peuvent facilement être intégrés dans les pages Google Sites.

Pour intégrer un lecteur Brightcove à une page Google Sites, procédez comme suit :

  1. Recherchez la page Google Sites à laquelle vous souhaitez ajouter un lecteur Brightcove.
  2. Clique le Modifier la page bouton.
  3. Une barre d'outils apparaît en haut de la page. Clique le <HTML> bouton.
  4. Collez le code de publication de votre lecteur Standard (iframe) et cliquez sur Mettre à jour.
  5. La page mettra à jour et affichera un gadget Google pour l'iframe.
  6. Cliquez sur Save pour enregistrer les modifications apportées. La vidéo sera affichée dans la page.
