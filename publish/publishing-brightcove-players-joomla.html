Joomla est un CMS open source populaire qui vous permet de concevoir des sites Web et des applications avancées en ligne. Vous trouverez plus d'informations sur Joomla sur cette page. Cette rubrique vous montre comment vous pouvez facilement publier Brightcove Players sur vos pages Joomla.

Note: Les étapes et les images de cette rubrique ont été prises à l'aide de Joomla 3.9.6.

Pour publier un lecteur Brightcove sur une page Joomla, procédez comme suit :

  • Vérifiez les paramètres de l'éditeur Joomla
  • Coller dans l'intégration du lecteur code

Vérification des paramètres de l'éditeur Joomla

Par défaut, Joomla utilise l'éditeur TinyMCE. Cet éditeur supprime certaines balises HTML lorsqu'elles sont copiées-collées. Pour coller le code de publication du lecteur Standard (iframe) dans l'éditeur Joomla, vous devez vous assurer que le <iframe> la balise ne sera pas supprimée par l'éditeur.

  1. Connectez-vous en tant qu'administrateur à votre interface de gestion Joomla.
  2. Sélectionner Extensions> Plugins.
  3. Entrer Tinymce dans la zone de recherche et cliquez sur le bouton de recherche.
  4. Vérifiez qu'un élément est renvoyé, Editeur - TinyMCE.
  5. Cliquez sur le lien Editor - TinyMCE plugin pour modifier ses paramètres.
  6. Localisez le Éléments interdits section et assurez-vous iframe n'est pas répertorié. Si iframe est répertorié, supprimez-le de la liste.
    Note: Joomla exige qu'il y ait une valeur dans le Éléments interdits champ. Si la suppression de l' iframe entrée laisse ce champ vide, entrez  aucun » dans ce champ.
  7. Cliquez sur sauver en haut de la page pour enregistrer vos modifications.

L'éditeur TinyMCE supprimera également le <script> lors de l'utilisation du code de publication du lecteur avancé (encart). Une façon de contourner cela est de configurer votre instance Joomla pour ne pas utiliser l'éditeur TinyMCE. Pour sélectionner un autre éditeur, procédez comme suit :

  1. Aller à Système > Configuration globale.
  2. Clique le Site languette.
  3. Trouvez le Éditeur par défaut option.
  4. Définissez cette option sur Editeur - Aucun. Nous avons également trouvé que la sélection Editeur - CodeMirror ne dépouillera pas la <script> balise.
  5. Cliquez sur Save & Close.

Si votre installation Joomla est configurée pour utiliser un autre éditeur, vous devez vous assurer que le code de publication du lecteur n'est pas modifié par l'éditeur lorsque vous collez et enregistrez le code dans Joomla. Vous devrez peut-être modifier d'autres paramètres Joomla ou éventuellement utiliser une extension (Jumi ou Sourcerer) pour vous permettre de coller du code JavaScript dans votre éditeur.

Coller dans le code d'intégration du lecteur

Une fois que votre éditeur est configuré de manière à ne pas modifier le code, vous pouvez générer le code d'intégration du lecteur.

  1. À l'aide du module Media, générez le code d'intégration de la vidéo que vous souhaitez publier.
  2. Ouvrez la page à éditer dans Joomla.
  3. Clique le Changer d'éditeur bouton en bas de l'éditeur pour passer en mode HTML.
  4. Collez votre code de publication et cliquez sur Save & Close (Enregistrer et fermer).
  5. Vérifiez que la vidéo s'affiche et qu'elle est lue sur votre page.
