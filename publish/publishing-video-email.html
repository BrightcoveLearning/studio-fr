Sur la plupart des plates-formes, l'intégration d'une vidéo directement dans un courriel est impossible. Il ne s'agit pas d'une restriction de Brightcove. Pour des raisons de sécurité, presque tous les systèmes de messagerie modernes suppriment le code d'intégration au lecteur vidéo. Vous pouvez cependant stimuler la sensibilisation et le trafic pour le contenu en créant un e-mail contenant du texte de teaser et une image d'affiche. Le module Media permet de publier une vidéo sur une page de destination d'un e-mail.

Note: Pour obtenir une autre option sur l'envoi de vidéos par e-mail qui permet de mieux contrôler l'apparence de l'image de l'affiche vidéo et d'utiliser une page de destination personnalisée, voir Driving Video Views Through Email.

Pour publier une vidéo sur une page de destination d'un e-mail, procédez comme suit :

  1. Ouvrez le module Médias.
  2. Sélectionnez une vidéo.
  3. Cliquez sur Publier et intégrer> Page de destination des e-mails.
  4. Sélectionnez une page de destination. La liste déroulante contiendra une page de destination simple qui est un système fourni, exemple de page de destination non stylé. La liste contient également toutes les expériences de page de destination publiées qui ont été créées à l'aide du module Galerie. L'utilisation d'une page de destination Galerie offre plus de flexibilité sur l'apparence et le style de la page de destination. Les formulaires de prospect peuvent également être utilisés pour capturer des informations sur les spectateurs. Pour plus d'informations sur la création de pages de destination de la Galerie, voir Création d'une expérience de page de destination.
    Note: L'image de l'affiche sera dimensionnée à 640 x 360.
  5. Sélectionnez Lecture automatique de la vidéo. Cette option n'est disponible que lorsque vous sélectionnez la page de destination simple.
  6. Sélectionnez un lecteur vidéo. Cette option n'est disponible que lorsque vous sélectionnez la page de destination simple. Pour les pages de destination de la Galerie, le lecteur est affecté dans le cadre des propriétés d'expérience.
  7. (Facultatif) Cliquez sur le Aperçu de la page de destination lien pour prévisualiser la page de destination.
  8. Mettez l'image en surbrillance, copiez-la dans le Presse-papiers, puis collez-la dans un message électronique.

Cliquez sur Générer un lien court pour générer un lien raccourci qui peut également être utilisé et distribué. Lorsque l'e-mail est reçu, cliquer sur l'image de l'affiche ou le lien court ouvrira la vidéo dans un onglet du navigateur.

Si une page de destination Gallery est utilisée avec un formulaire pour prospects, les données du formulaire pour prospects peuvent être téléchargées à l'aide du module Campagne. Pour plus d'informations, voir Exportation des données de prospect de la galerie
