Les codes UTM (également appelés paramètres UTM) sont de simples extraits de texte que vous pouvez ajouter à une URL pour vous aider à suivre le succès de votre contenu Web. Il y a 5 paramètres standard que vous pouvez ajouter à n'importe quelle URL :

  • Campagne
  • Source
  • Moyenne
  • Contexte
  • Terme

Les codes UTM sont généralement ajoutés aux URL personnalisées. Voici un exemple de codes UTM (en gras) ajoutés à une URL.

https://www.brightcove.com/en/blog/2018/08/define-your-video-strategy-3-simple-steps-1?utm_campaign=blogpost&utm_medium=social&utm_source=LinkedIn

Les codes UTM vous permettent de suivre le fonctionnement d'un billet de blog, d'une annonce Internet ou d'un message électronique sans avoir à créer des pages de destination personnalisées pour chaque campagne. En créant un code UTM distinct pour les billets de blog, les annonces et les e-mails, vous pouvez obtenir des données sur lesquelles génère plus de trafic, de conversions, etc.

Configuration des connexions de données pour travailler avec les champs UTM

Campaign analyse les champs UTM à partir des arguments d'URL. Les données UTM sont ensuite accessibles à partir de votre logiciel d'automatisation marketing ou à l'aide de l'API Audience. La configuration de la campagne requise est basée sur la plate-forme d'automatisation du marketing utilisée.

  • HubSpot
  • Salesforce
  • Éloqua
  • Marketo

HubSpot

Avec les connexions de données HubSpot, aucune configuration supplémentaire n'est requise. Les champs UTM apparaîtront automatiquement.

Force de vente

Avec les connexions de données Salesforce, utilisez un Type de mappage de Avancée puis les champs UTM peuvent être mappés aux objets de données appropriés dans Salesforce.

Éloqua

Avec les connexions de données Eloqua, aucune configuration supplémentaire n'est requise. Les champs UTM apparaîtront automatiquement.

Marketo

Étant donné qu'il n'y a pas suffisamment de champs que nous pouvons ajouter à l'activité personnalisée que nous créons dans Marketo, les champs UTM ne sont pas pris en charge avec les connexions Marketo.

Exporter des données à l'aide du module Campaign

Les champs UTM sont inclus lors de l'exportation des données de la campagne et de la sélection Afficher les événements. Pour plus d'informations sur l'exportation de données, voir Exportation des données de campagne à partir du module de campagne.

Accès aux champs UTM à l'aide de l'API Audience

Les champs UTM sont accessibles à l'aide de l'API Audience. Pour plus d'informations, consultez la documentation de l'API d'audience.
