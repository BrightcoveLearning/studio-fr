Lors de l'utilisation des intégrations de suivi de campagne, des profils de visionneuse seront créés en fonction des données qui ont été utilisées pour identifier la visionneuse. Les profils de visionneuse vous permettent d'afficher les données d'affichage d'une visionneuse donnée.

Toutes les vues vidéo dans les lecteurs activés pour la campagne seront suivies. Pour afficher l'activité vidéo, cliquez sur Profil du visualiseur (non-carte) dans la navigation de gauche. La page Profils des spectateurs affichera une liste de toutes les activités de visualisation des intégrations de suivi de campagne pour la période sélectionnée. Le contrôle calendrier peut être utilisé pour sélectionner une nouvelle plage de dates. La valeur Viewer est la valeur utilisée pour identifier l'utilisateur. En règle générale, ce sera l'adresse e-mail capturée sur un formulaire de prospect. Les données peuvent être triées en cliquant sur les en-têtes de colonne.

Note: Il y aura quelques heures de retard avant que les données d'affichage ne s'affichent.

Cliquez sur le lien d'une visionneuse pour afficher toute l'activité de visualisation de cette visionneuse. Les données peuvent être triées en cliquant sur les en-têtes de colonne.

Les données peuvent être exportées en cliquant sur les boutons CSV ou XLS en haut de la page.

Autres options de visualisation des données

Les données de campagne peuvent être visualisées/exportées d'autres manières :

  • Activité récente: permet de visualiser les données d'événements de visionnage vidéo au cours des dernières 24 heures. Pour plus d'informations, voir Affichage des événements de lectuer dans le module de campagne.
  • Activité vidéo: permet de visualiser l'activité de la visionneuse par vidéo. Pour plus d'informations, voir Affichage de l'activité vidéo.
  • Exporter des données: permet d'exporter des données de vue vidéo et de prospect. Pour plus d'informations, voir Exportation des données de campagne à partir du module de campagne.
  • API Audience - L'API Audience peut être utilisée pour récupérer par programmation des vues vidéo et des données de piste. Pour plus d'informations, consultez Généralités: API d'audience.
