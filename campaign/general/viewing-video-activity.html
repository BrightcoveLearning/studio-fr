Lorsque vous utilisez le suivi de campagne ou les intégrations SSO, le module Campagne vous permet de sélectionner une vidéo, puis d'afficher l'activité de cette vidéo. Cela vous permet de voir qui a regardé une vidéo particulière.

Toutes les vues vidéo dans les lecteurs activés pour la campagne seront suivies. Pour afficher l'activité vidéo, cliquez sur Vidéo (non-carte) dans la navigation de gauche. La page Activité de visionnage affichera une liste de toutes les vidéos visionnées à partir des intégrations de suivi SSO et de campagne pour la période sélectionnée. Le contrôle Sélectionner une connexion permet de sélectionner une connexion spécifique. Le contrôle calendrier peut être utilisé pour sélectionner une nouvelle plage de dates. Les données peuvent être triées en cliquant sur les en-têtes de colonne.

Notes :
  • Il y aura un délai de quelques heures avant que les données de visualisation ne s'affichent.
  • Analyse et suivi de campagne via des plugins indépendants et des systèmes backend. Il est très fréquent de ne pas voir les mêmes résultats. Campaign adhère également à des choses comme ne pas suivre les paramètres dans le navigateur. Pour obtenir les analyses les plus complètes, consultez les rapports du module Analytics.

Cliquez sur le lien d'une vidéo pour afficher toute l'activité de visualisation de cette vidéo. Les données peuvent être triées en cliquant sur les en-têtes de colonne.

Les données peuvent être exportées en cliquant sur les boutons CSV ou XLS en haut de la page.

Autres options de visualisation des données

Le module Campaign propose plusieurs manières d'afficher/exporter les données :

  • Activité récente: permet de visualiser les données d'événements de visionnage vidéo au cours des dernières 24 heures. Pour plus d'informations, voir Affichage des événements de lectuer dans le module de campagne.
  • Profils de visionneuse: permet de visualiser l'activité vidéo par visionneuse. Pour plus d'informations, voir Affichage des informations de profil de la visionneuse
  • Exporter des données: permet d'exporter des données de vue vidéo et de prospect. Pour plus d'informations, voir Exportation des données de campagne à partir du module de campagne.
  • API Audience - L'API Audience peut être utilisée pour récupérer par programmation des vues vidéo et des données de piste. Pour plus d'informations, consultez Généralités: API d'audience.
