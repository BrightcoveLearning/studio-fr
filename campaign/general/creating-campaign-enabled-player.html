Les données d'engagement vidéo ne seront enregistrées que lorsque le contenu vidéo est visionné dans un Brightcove Player activé pour la campagne. Il existe deux manières de créer un lecteur activé pour la campagne :

  • Créer un nouveau lectuer à l'aide du module Campagne
  • Activer Campagne sur un lecteur existant

Créer un nouveau lectuer à l'aide du module Campagne

Avant de créer un lecteur compatible avec Campaign à l'aide du module Campaign, le module Campaign doit être configuré avec au moins une connexion de données :

  • Eloqua
  • Marketo (Munchkin)
  • Marketo (API REST)
  • Salesforce
  • HubSpot (API REST)
  • HubSpot (API côté client)
  • SSO
  • Intégration personnalisée

Noter que Suivi de campagne est fourni par défaut. Pour créer un lecteur compatible avec la campagne à l'aide du module Campagne, procédez comme suit.

  1. Ouvrez le module Campagne.
  2. Cliquez sur lectuers de la campagne dans la navigation de gauche.
  3. Cliquez sur Créer un lecteur. Un nouveau lectuer activé pour la campagne sera créé et apparaîtra dans la liste des lectuers.
  4. En utilisant le Se connecter à liste déroulante, sélectionnez la connexion de données que le lecteur utilisera.
  5. (Optionnel) En utilisant le Formulaire de prospect liste déroulante, sélectionnez le formulaire de plomb que le lectuer utilisera.

Activation de Campaign sur un lecteur existant

Pour activer la campagne d'un lecteur existant, procédez comme suit.

  1. Ouvrez le module Players.
  2. Cliquez sur le nom d'un lecteur pour ouvrir ses propriétés.
  3. Cliquez sur Campagne dans le menu de navigation de gauche.
  4. Vérifier la Activer la campagne option.
  5. Utilisez le Se connecter à liste déroulante pour sélectionner une connexion à la campagne.
  6. (Facultatif) Utilisez la liste déroulante Formulaire de prospect pour sélectionner un formulaire de prospect à utiliser.
  7. Cliquez sur Save. Un message s'affiche indiquant que vous avez des modifications non publiées.
  8. Cliquez sur Publier et intégrer, puis sur Publier les modifications pour publier la modification sur le lecteur.
  9. Retournez au module Campagne. Cliquez sur lectuers de la campagne dans la navigation de gauche.
  10. Confirmez que le lectuer apparaît dans la liste des lectuers de la campagne.
