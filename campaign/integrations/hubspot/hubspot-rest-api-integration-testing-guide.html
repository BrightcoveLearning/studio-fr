Lors de la configuration d'une nouvelle intégration HubSpot à l'aide des API REST, il peut être difficile de tester et de confirmer que les données de visualisation vidéo sont effectivement fournies à HubSpot. Dans cette rubrique, vous apprendrez comment tester et confirmer systématiquement que l'intégration fonctionne et que l'affichage des données et des prospects sont envoyés à HubSpot.

Note: Cette rubrique concerne la vérification des connexions HubSpot effectuées à l'aide des API REST.
Note: Pour plus d'informations sur la configuration du module Campaign pour une utilisation avec HubSpot, consultez la section Intégration de Video Cloud avec HubSpot à l'aide des API REST.

Préliminaires

Avant de valider l'intégration, il est supposé que vous avez :

  • A terminé avec succès le Intégration de Brightcove Campaign avec HubSpot
  • créé au moins un Lecteur activé pour la campagne et configuré pour utiliser la connexion HubSpot

Configuration d'un formulaire de prospect de test et publication d'une vidéo

À des fins de test, vous devrez créer et configurer un formulaire pour prospects personnalisé dans Campaign. Il s'agit d'un formulaire HubSpot dont vous avez collé le code d'intégration dans Campaign. Même si vous n'utiliserez pas de formulaires de prospect dans votre environnement de production, ce formulaire est utilisé pour vous assurer que vous êtes connu dans votre base de données et sur la page. Pour créer un formulaire HubSpot et configurer Campaign pour l'utiliser, procédez comme suit.

  1. Connectez-vous à votre compte HubSpot.
  2. Créez et stylisez un formulaire HubSpot. Pour plus d'informations sur la création d'un formulaire HubSpot, voir Création de formulaires de prospect personnalisés pour HubSpot.
  3. Cliquez sur Partager en haut de la page.
  4. Copiez le code d'intégration du formulaire dans le Presse-papiers.
  5. Retournez au module Campagne.
  6. Créez un nouveau formulaire de prospect à l'aide du code d'intégration du formulaire HubSpot.
  7. Définissez votre formulaire personnalisé pour afficher le formulaire à la marque de 5 secondes. De cette façon, vous saurez exactement quand il devrait apparaître.
  8. Cochez la case à Toujours afficher le formulaire prospect même si le prospect est déjà connu. Cela garantira que le formulaire continue de s'afficher après l'avoir rempli pendant le test.
  9. Enregistrez le formulaire.
  10. Configurez Brightcove Player avec la connexion HubSpot pour utiliser le formulaire HubSpot qui vient d'être créé.

Génération d'un lien d'aperçu pour visionner la vidéo

Pour générer un lien d'aperçu vers la vidéo, procédez comme suit.

  1. Ouvrez le module Médias.
  2. Sélectionnez une vidéo et publiez-la sur un lecteur Web. Assurez-vous de sélectionner le lectuer activé pour la campagne avec le formulaire de prospect.
  3. Cliquez sur le lien de prévisualisation pour ouvrir la vidéo dans une nouvelle fenêtre de navigateur. Le test de cette façon élimine toutes les variables potentielles qui pourraient être introduites par les lectuers intégrés sur une page HTML.
  4. Confirmez que la vidéo est lue et que le formulaire d'avance s'affiche à la barre de 5 secondes.
  5. Remplissez le formulaire de prospect en utilisant des informations de contact uniques et une adresse e-mail (c.-à-d. un nom de célébrité ou un nom contenant les mots « bctest ») afin qu'il puisse être facilement trouvé plus tard dans HubSpot. Cela élimine toutes les variables de niveau navigateur, lectuer ou base de données. En remplissant le formulaire en tant que nouvel utilisateur, nous pouvons être sûrs que vous êtes maintenant un utilisateur connu dans HubSpot.

Confirmation que Campaign capture les données avec succès

Avant de rechercher des données de visionnage de vidéo dans HubSpot, vérifiez d'abord que Campaign a bien capturé la vue vidéo. Notez que l'affichage des données peut prendre jusqu'à 3 heures pour s'afficher dans l'interface utilisateur de la campagne. Pour confirmer que Campaign capture correctement les données, procédez comme suit.

  1. Ouvrez le module Campagne.
  2. Cliquez sur Toutes les activités récentes dans la navigation de gauche.
  3. Confirmez qu'il existe un événement avec un horodatage récent indiquant le nom de la vidéo qui vient d'être regardée. L'état de l'événement doit être non traité. Cela indique que Campaign a capturé la vue mais n'a pas encore tenté de la synchroniser avec HubSpot.
  4. Pour déclencher manuellement une synchronisation entre Campaign et HubSpot, cliquez sur Activité de synchronisation dans la navigation de gauche.
  5. Dans la liste déroulante Sélectionner une connexion, sélectionnez votre intégration HubSpot. Un bouton Test Sync devrait s'afficher.
  6. Cliquez sur Test Sync pour déclencher une synchronisation manuelle. Notez que vous êtes limité à la fréquence à laquelle vous pouvez déclencher une synchronisation manuelle. Par défaut, Campaign synchronisera les vues avec HubSpot une fois par heure.
  7. Cliquez sur Toutes les activités récentes dans la navigation de gauche.
  8. L'état de votre événement view sera mis en file d'attente. La synchronisation des données sur HubSpot peut prendre environ 15 minutes. Une fois les données synchronisées, l'état passe à l'un des deux statuts suivants :
    • synced - Indique que l'utilisateur était un utilisateur connu et que les données ont été correctement livrées dans son journal d'activité. Les données doivent également être visualisées dans HubSpot en quelques minutes.
    • unknown_contact - Indique que l'utilisateur était considéré comme inconnu et que HubSpot n'a pas pu accepter les données. Si vous obtenez un contact_inconnu statut, vérifiez que vous avez publié la vidéo à l'aide du bon lecteur, que vous avez un formulaire HubSpot en place et que vous testez à l'aide du lien de prévisualisation, non intégré dans une page.

Confirmation que les données de la vue vidéo sont dans HubSpot

Une fois que Campaign a affiché un synchronisé statut, suivez ces étapes pour vérifier que les données sont dans HubSpot.

Note: Il faut 5 à 10 minutes pour que l'interface utilisateur HubSpot soit mise à jour avec les nouvelles données qu'elle a reçues de Campaign.
  1. Connectez-vous à votre compte HubSpot.
  2. Dans HubSpot, accédez à la zone Contacts et recherchez le nom unique ou l'adresse e-mail que vous avez utilisé lors du remplissage du formulaire de prospect.
  3. Cliquez sur le contact pour ouvrir le journal d'activité.
  4. Clique sur le Filtrer l'activité liste déroulante (en haut de la page sous le Activité menu) et confirmez que Connexion vidéo Brightcove est vérifié. Cela garantit que l'activité de visionnage vidéo est affichée dans le journal d'activité.
  5. Dans le journal d'activité, recherchez un Brightcove activité. Développez les détails pour afficher les données de l'événement de visualisation de la vidéo.

HubSpot exige que les données fournies via leur API soient attribuées à un contact connu. HubSpot acceptera uniquement les données des contacts connus. Les données des contacts inconnus ne seront pas synchronisées. Si vous n'utilisez pas de formulaires de prospect, il n'y a aucun moyen pour un utilisateur inconnu de se convertir en utilisateur connu et donc aucune donnée ne sera synchronisée.
