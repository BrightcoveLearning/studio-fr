Le module Campaign offre la possibilité de créer des formulaires pour prospects qui vous permettent de capturer des informations sur les téléspectateurs pendant que les vidéos sont lues dans les lecteurs Brightcove. Vous avez ainsi la possibilité d'inclure les champs dont vous avez besoin (visibles ou masqués) et de tirer parti de fonctionnalités telles que le profilage progressif (si disponible). Les données de prospect sont envoyées à Marketo et traitées par Marketo.

La création d'un formulaire de clients potentiels passe par les étapes suivantes :

  1. Créer un formulaire
  2. Facultatif - Style du formulaire à l'aide de CSS
  3. Copier le code d'intégration du formulaire
  4. Créer un formulaire pour prospects personnalisé dans le module Campagne
  5. Attribuer le formulaire de prospection à un lectuer
  6. Valider les données du formulaire dans Marketo
Note: Pour plus d'informations sur les formulaires de prospects standard qui capturent les informations de visionneuse et enregistrent les données dans Video Cloud, voir Création d'un formulaire de pistes de campagne.

Création d'un formulaire

Marketo offre la possibilité de créer des formulaires pour capturer des informations de visionneuse. Ces formulaires peuvent être affichés dans les lecteurs Brightcove activés pour Campaign. Les données capturées sont ensuite utilisées à des fins de personnalisation pour les communications ultérieures. La procédure ci-dessous permet de créer un formulaire simple pour capturer le prénom, le nom et l'adresse électronique. Pour plus de détails sur les formulaires Marketo, reportez-vous à la documentation Marketo.

  1. Connectez-vous à votre compte Marketo.
  2. Accédez à Design Studio > Formulaires, puis cliquez sur Nouveau > Nouveau formulaire. La boîte de dialogue Nouveau formulaire s'ouvre.
  3. Entrez un nom pour le formulaire et cliquez sur CRÉER.
  4. Le concepteur de formulaire s'ouvrira. Par défaut, les champs Prénom, Nom et Adresse e-mail seront sur le formulaire. Modifiez les propriétés et le comportement du champ selon vos besoins. Cliquez sur SUIVANT.
  5. Cliquez sur les icônes fléchées pour sélectionner un thème de formulaire. Le thème du formulaire contrôle le style du formulaire. Cliquez sur Prévisualiser le brouillon pour afficher un aperçu du formulaire. Dans cet exemple, le thème Dark a été sélectionné.
  6. Cliquez sur Modifier le brouillon pour revenir à l'éditeur de formulaire.
  7. Cliquez sur SUIVANT.
  8. Configurez les paramètres de formulaire supplémentaires et cliquez sur TERMINER.
  9. Cliquez sur APPROUVER ET FERMER.

Facultatif - Styler le formulaire à l'aide de CSS

Lors de la création d'un formulaire, Marketo permet de sélectionner un thème de formulaire pour fournir un style de formulaire de base. Vous pouvez également personnaliser le formulaire en utilisant votre propre CSS personnalisé. Dans cette section, nous allons personnaliser le thème Dark et rendre le texte de l'étiquette blanc et gras. Pour plus de détails sur les formulaires Marketo, reportez-vous à la documentation Marketo.

  1. Accédez à Design Studio > Formulaires, puis cliquez sur votre formulaire.
  2. Cliquez sur Modifier le formulaire.
  3. Cliquez sur Paramètres de formulaire, puis sur l'icône en forme de roue dentée.
  4. Cliquez sur Afficher le thème CSS pour afficher le thème CSS actuel. Cliquez sur Modifier le CSS personnalisé. Collez le code CSS suivant pour que le texte de l'étiquette soit blanc et gras. 
    étiquette .mktoForm { font-weight: bold; color: #FFF; }
    Note: Entrez uniquement votre CSS personnalisé (juste les parties qui sont différentes). Vous n'avez pas à coller dans l'intégralité du thème CSS.
  5. Cliquez sur SAVE.
  6. Cliquez sur Prévisualiser le brouillon pour afficher un aperçu du formulaire. Le texte de l'étiquette doit être blanc et gras.
  7. Cliquez sur Modifier le brouillon pour revenir à l'éditeur de formulaire.
  8. Cliquez sur TERMINER , puis APPROUVER ET FERMER.

Copie du code d'intégration du formulaire

Pour utiliser le formulaire à l'intérieur du module Campaign, vous devrez copier le code d'intégration du formulaire.

  1. Dans la navigation de gauche, faites un clic droit sur le formulaire et cliquez sur Code intégré.
  2. Met le Type de code à Normal puis copiez le Code au presse-papiers. Cliquez sur Close (Fermer).

Création d'un formulaire pour prospects personnalisé dans le module Campagne

Une fois le formulaire créé et stylisé, créez un formulaire pour prospects personnalisé dans le module Campagne et associez le formulaire pour prospects à un lecteur compatible Campagne.

  1. Ouvrez le module Campagne.
  2. Cliquez sur Formulaires de prospect dans la navigation de gauche. Si des formulaires de clients potentiels ont déjà été créés, ils s'afficheront ici.
  3. Cliquez sur Create Lead Form.
  4. Donnez au formulaire un nom descriptif.
  5. Choisissez la langue dans Form Language.
  6. Sélectionnez le moment d'affichage dans Form Timing. Cela permet de déterminer quand s'affichera le formulaire de clients potentiels.
  7. Définissez le type de formulaire sur Custom Lead Form.
  8. Coller dans le formulaire HTML à partir de Marketo.
  9. (Facultatif) Collez dans le code HTML de confirmation. La vue de confirmation s'affiche pendant 5 secondes, puis la vidéo reprendra la lecture. Si la confirmation HTML n'est pas incluse, la lecture de la vidéo reprendra après l'envoi du formulaire.
  10. (Facultatif) Cliquez sur Preview Form pour afficher un aperçu du formulaire.
  11. Cliquez sur Save pour enregistrer le formulaire.

Affectation d'un formulaire de clients potentiels à un lecteur

Une fois le formulaire de client potentiel créé, vous devez l'affecter à un lecteur. Pour ce faire, procédez comme suit :

  1. Cliquez sur lectuers de la campagne dans la navigation de gauche.
  2. Localisez votre lectuer et utilisez la liste déroulante Formulaire de prospect pour sélectionner le formulaire de prospect.

La configuration est désormais terminée ; vous êtes prêt à capturer les informations sur les clients potentiels. Lorsqu'une vidéo est publiée à l'aide d'un lecteur compatible avec Campaign, le formulaire pour prospects apparaîtra en fonction des paramètres de synchronisation du formulaire.

Validation des données de formulaire dans Marketo

Il existe plusieurs façons de valider que les données du formulaire ont été transmises à Marketo. Procédez comme suit pour afficher les données du formulaire dans Marketo.

  1. Connectez-vous à votre compte Marketo.
  2. Cliquez sur Base de données prospect.
  3. Dans la navigation de gauche, développez Listes dynamiques de groupe, puis cliquez sur Form Capture SL.
  4. Clique le Liste intelligente languette.
  5. Dans la liste des Filtres sur le côté droit de la page, localisez le Formulaire rempli filtre. Cliquez sur le filtre et faites-le glisser vers la page.
  6. Entrez le nom de votre formulaire. Vous pouvez également cliquer sur le + pour rechercher votre formulaire.
  7. Cliquez sur Responses en haut de la page pour afficher les données de prospect filtrées.
  8. Cliquez sur un ID pour ouvrir le détail du prospect.
  9. Clique le Journal d'activité languette. Confirmer que le formulaire a été rempli.
