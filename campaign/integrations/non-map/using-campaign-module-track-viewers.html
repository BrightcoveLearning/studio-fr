Le module Campagne offre la possibilité de suivre l'activité de visualisation. Dans ce scénario, l'affichage des données n'est pas envoyé à une plate-forme d'automatisation marketing. Un cas d'utilisation courant pour cela serait les vidéos de formation internes. Le suivi de campagne peut être utilisé pour suivre l'activité de visualisation des employés internes. Pour utiliser le suivi de campagne, les étapes suivantes sont requises :

  1. Configurer les paramètres de suivi de campagne
  2. Configurer un formulaire de lectuer et de prospect pour le suivi de campagne
  3. Afficher les données vidéo

Configuration des paramètres de suivi de campagne

Pour configurer les paramètres de suivi de campagne, procédez comme suit :

  1. Ouvrez le module Campagne puis cliquez sur Connexions de données dans la navigation de gauche.
  2. Localisez le Suivi de campagne rubrique et cliquez sur Spectacle.
  3. Par défaut, le suivi des campagnes est activé pour tous les comptes. Modifiez les paramètres selon vos besoins :
    • Nom de la connexion: identifie la connexion
    • Statut de connexion - Détermine si les données d'engagement vidéo de Video Cloud sont enregistrées par le module Campaign.
      • Activé - Les données sont collectées
      • Désactivé - Aucune donnée n'est collectée (la connexion est désactivée)
    • Suivi des données pour les visionneuses inconnues: détermine si les données de suivi sont enregistrées pour les visionneuses inconnues. Pour identifier les spectateurs lors de l'utilisation du suivi de campagne, un Formulaire de prospect de campagne peut être utilisé.
      Note: Avec le suivi des campagnes, jusqu'à 1 000 spectateurs seront suivis par période de 24 heures.
    • Combien de temps Brightcove doit-il conserver les données ? - La politique de conservation des données détermine la durée pendant laquelle les données de visualisation et les données du formulaire de prospect (si vous utilisez un formulaire de prospect Campaign) seront conservées dans votre compte Video Cloud. Vous devez ajuster les paramètres de rétention conformément à votre politique de confidentialité. Il n'est pas nécessaire de conserver des données utilisateur dans Video Cloud.

Configuration d'un lectuer et d'un formulaire de prospect pour le suivi de campagne

L'activité de visionnage ne peut être suivie que lorsque les vidéos sont visionnées dans un lecteur vidéo compatible avec Campaign. Vous devez utiliser un lecteur compatible avec la campagne lorsque vous utilisez le suivi de campagne. Les formulaires de prospect peuvent être affichés pendant la lecture vidéo pour capturer les informations du spectateur. Les formulaires pour prospects ne sont pas requis lors de l'utilisation du suivi de campagne.

Pour configurer le suivi de campagne avec un lecteur et un formulaire pour prospects activés pour la campagne, procédez comme suit :

  1. Cliquez sur lectuers de la campagne dans la navigation de gauche.
  2. Déterminez quel lecteur compatible avec la campagne utiliser. S'il n'existe aucun lecteur activé pour la campagne, cliquez sur Créer un lectuer pour en créer un. Pour plus d'informations sur la création de lecteurs activés pour la campagne, consultez Création d'un lecteur compatible avec la campagne.
  3. Bien qu'un formulaire de prospect n'est pas requis, les formulaires de prospect peuvent être utilisés pour recueillir des informations sur le spectateur et transformer les spectateurs anonymes en spectateurs connus. Cliquez sur Formulaires de prospect et créez un formulaire de prospect. Pour plus d'informations sur la création de formulaires pour prospects, consultez Création d'un formulaire de prospect de campagne.
  4. Cliquez sur lectuers de la campagne.
  5. Localisez le lecteur compatible avec la campagne à utiliser. Met le Se connecter à valeur pour le lectuer à Suivi de campagne.
  6. Si un formulaire pour prospects est utilisé, sélectionnez-le dans le Formulaire de prospect la liste déroulante.

Affichage des données vidéo

Plusieurs options sont disponibles pour afficher les données vidéo lors de l'utilisation du suivi de campagne :

  • Activités des spectateurs (événements) - Fourni dans le cadre du module Campagne, cette option peut être utilisée pour afficher les données vidéo à l'intérieur du module Campagne. Pour plus d'informations, voir Affichage des événements de lectuer dans le module de campagne.
  • Profils de visionneuse - Fourni dans le cadre du module Campagne, cette option peut être utilisée pour afficher des données vidéo en fonction d'une valeur de profil utilisateur à l'intérieur du module Campagne. Pour plus d'informations, voir Affichage des informations de profil de la visionneuse
  • Exporter des données - Fourni dans le cadre du module Campagne, cette option peut être utilisée pour exporter la vue vidéo et les données de prospect. Pour plus d'informations, voir Exportation des données de campagne à partir du module de campagne.
  • API Audience - L'API Audience peut être utilisée pour récupérer par programmation des vues vidéo et des données de piste. Pour plus d'informations, consultez Généralités: API d'audience.
