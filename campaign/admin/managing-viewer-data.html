L'option Gérer les données de la visionneuse vous permet de rechercher un utilisateur, puis de supprimer cet utilisateur ou d'exporter les données de cet utilisateur.

Notes :
  • Généralement, lors de l'intégration à une plate-forme d'automatisation du marketing (intégration MAP), le paramètre de période de conservation des données est utilisé pour contrôler la durée de conservation des données.
  • Vous devez être un administrateur de compte pour accéder au menu Gérer les données de la visionneuse

Pour gérer les données de la visionneuse, procédez comme suit.

  1. Ouvrez le module Campagne.
  2. Cliquez sur Gérer les données du visualiseur dans la navigation de gauche.
  3. Entrez l'utilisateur à rechercher. Généralement, il s'agit de l'une des valeurs suivantes :
    • E-mail - Lorsque vous utilisez le suivi de campagne et qu'un formulaire de prospect de campagne est utilisé ou avec des intégrations SSO
    • ID utilisateur - Lors de l'utilisation de l'authentification unique ou d'intégrations personnalisées
    • GUID personnalisé - Pour les utilisateurs anonymes
    • GUID externe: lorsqu'une intégration MAP est utilisée
  4. Si une correspondance est trouvée, sélectionnez une action :
    • Supprimer les données de visionneuse - Supprime les données d'affichage et de prospect pour l'utilisateur
    • Exporter les données de visionneuse - Exportez les données des événements prospect et affichez l'utilisateur. Les données de prospect contiendront les données qui ont été saisies dans un formulaire de prospect de campagne. Les données d'événement de vue contiennent tous les événements de vue de la visionneuse.
