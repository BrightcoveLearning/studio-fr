Mise en route

Video Cloud Studio vous permet de gérer votre flux de travail de publication vidéo sur les ordinateurs de bureau, tablettes et appareils mobiles. Il offre une interface facile à utiliser pour télécharger, gérer et publier vos vidéos. Cette page décrit les concepts clés pour vous aider à comprendre et à réussir à utiliser Video Cloud Studio pour gérer votre flux de travail vidéo.
