Limites d'essai

Les comptes d'évaluation de Video Cloud présentent les limitations suivantes :

  • Un maximum de cinq vidéos peuvent être téléchargées sur un compte. Si vous avez besoin de pouvoir télécharger plus de 5 vidéos pendant que le compte est en mode d'essai, vous pouvez contactez notre équipe commerciale et ils vous fourniront un accès illimité pour le téléchargement. Si vous travaillez sur une intégration de partenaire Brightcove, veuillez contacter directement notre équipe de partenaires.
  • Pas d'accès aux profils d'ingestion
  • Le partage de médias n'est pas disponible
  • Les téléchargements d'images ne sont pas disponibles dans le module Galerie
  • Pas d'accès à la gestion des API
