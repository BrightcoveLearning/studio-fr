Conditions préalables

  • Vous avez téléchargé des vidéos sur votre compte Video Cloud
  • Vous avez ajouté quelques balises à vos vidéos

Étapes

  1. Ouvrez le module Médias.
  2. Dans le volet de navigation latéral, cliquez sur le lien Create Playlist.
  3. Définissez Playlist type sur Smart.
  4. Met le Nom de la playlist à Birds ou un autre nom approprié.
  5. Cochez l'option Show Playlist in Sidebar pour faire apparaître la sélection dans la barre latérale.
  6. Met le Mots clés champ à contient un ou plusieurs et entrez birds pour l'étiquette (ou quelle que soit la balise appropriée).
  7. Cliquez sur Save.
  8. Clique le le nom de la playlist vient d'être créé dans le panneau de navigation latéral et confirmez que les bonnes vidéos ont été ajoutées à la playlist.
