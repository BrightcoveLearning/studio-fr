Conditions préalables

  • Vous avez téléchargé des vidéos sur votre compte Video Cloud

Étapes

  1. Ouvrez le module Médias.
  2. Dans le volet de navigation latéral, cliquez sur le lien Create Playlist.
  3. Définissez Playlist type sur Manual.
  4. Entrez un Nom de la playlist.
  5. Cochez l'option Show Playlist in Sidebar pour faire apparaître la sélection dans la barre latérale.
  6. Cliquez sur Save.
  7. Pour ajouter des vidéos à la playlist, cliquez sur Toutes les vidéos dans le panneau de navigation latéral pour afficher toutes les vidéos du compte.
  8. Cochez les cases correspondantes aux vidéos à ajouter à la sélection.
  9. Cliquez sur Plus> Ajouter à la playlist.
  10. Sélectionnez une liste de lecture, puis cliquez sur Ajouter.

Pour réorganiser l'ordre des vidéos, sélectionnez la liste de lecture, puis cliquez sur une ligne et faites-la glisser vers une nouvelle position dans la liste.
