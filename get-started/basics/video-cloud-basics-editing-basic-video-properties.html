Conditions préalables

  • Vous avez téléchargé des vidéos sur votre compte Video Cloud

Étapes

  1. Dans Video Cloud Studio, cliquez sur Médias pour ouvrir le Médias module.
  2. Cliquez sur un titre vidéo pour ouvrir la page des propriétés de la vidéo.
  3. Les propriétés de la vidéo sont organisées en sections. Pour modifier les propriétés vidéo, cliquez sur Éditer ou double-cliquez dans une section.
  4. Cliquez sur Save pour enregistrer les modifications apportées.
