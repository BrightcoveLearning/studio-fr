Conditions préalables

  • Vous avez téléchargé des vidéos sur votre compte Video Cloud
  • Vous avez créé un Manuel ou Intelligent playlist

Étapes

  1. Ouvrez le module Médias.
  2. Sélectionnez une liste de lecture à publier :
    • Si la liste de lecture apparaît dans la navigation latérale, cliquez dessus, puis cliquez sur Publier la playlist
    • Cliquez sur Toutes les listes de lecture dans le menu de navigation de gauche, sélectionnez une playlist puis cliquez sur Publier et incorporer> Publier sur le Web
  3. Dans le Sélectionner un lectuer liste déroulante, sélectionnez un lecteur à utiliser.
  4. Cliquez sur l' URL située sous le lecteur de prévisualisation pour l'ouvrir dans un nouvel onglet de navigateur. Vérifiez que la liste de lecture s'affiche et que les vidéos peuvent être lues.
  5. Copiez le code d'intégration et collez-le dans votre page Web.
