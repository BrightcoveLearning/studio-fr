Étapes

  1. Dans Video Cloud Studio, ouvrez le Analytique module.
  2. Cliquez sur Engagement dans la navigation de gauche.
  3. Pour spécifier la période du rapport, utilisez le sélecteur de date en haut de la page.
  4. Sélectionnez pour afficher le Engagement par Vidéo dimension.
  5. Clique sur le Vues à 1% colonne pour trier les données.
  6. Cliquez sur un titre vidéo pour afficher les données d'engagement de cette vidéo. Le graphique montre le nombre de vues vidéo au fil du temps. Les pointes dans le graphique indiquent qu'une partie de la vidéo a été regardée encore et encore. Les chutes soudaines d'engagement devraient également faire l'objet d'une enquête.
  7. Cliquez sur la ligne d'engagement pour visualiser cette partie de la vidéo dans le lecteur d'aperçu.

Le champ de recherche en haut de la page peut être utilisé pour rechercher une vidéo spécifique.
