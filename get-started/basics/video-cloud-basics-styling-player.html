Conditions préalables

  • Vous avez a créé un lectuer dans votre compte Video Cloud

Étapes

  1. Ouvrez le module Players.
  2. Cliquez sur le nom du lectuer à styliser.
  3. Cliquez sur Coiffant dans la navigation de gauche.
  4. Choisissez d'afficher ou de masquer le titre et description.
  5. Sélectionner un Forme du bouton de lecture et Position du bouton de lecture.
  6. Cliquez sur une couleur de thème pour ouvrir le sélecteur de couleurs.
  7. Cliquez sur une couleur sur le côté droit.
  8. Cliquez sur le carré pour sélectionner la nuance de la couleur voulue.
  9. Le curseur situé sous le carré de couleur peut être utilisé pour définir l'opacité des contrôles.
  10. Cliquez en dehors du sélecteur de couleurs pour modifier une autre couleur de thème.
  11. Modifications apportées à un lectuer DOIT seront publiés avant d’apparaître dans les lecteurs publiés. Pour publier les modifications, cliquez sur Publier et intégrer et alors Publier les modifications.
