Pour accéder à Video Cloud Studio, accédez à: https://studio.brightcove.com. Connectez-vous en indiquant votre adresse électronique et votre mot de passe. La page d'accueil offre la possibilité de :

  • Afficher les notifications système
  • Gérer les paramètres de Video Cloud
  • Sélectionnez un compte et affichez le profil utilisateur
  • Accéder aux modules Video Cloud
  • Afficher le tableau de bord de Video Cloud

Vous pouvez avoir un maximum de 5 sessions à tout moment. Une session est lancée chaque fois que vous vous connectez avec la même adresse e-mail. Une fois qu'une 6e session est lancée, la session la plus ancienne est invalidée. Le délai d'expiration de la session Video Cloud Studio est de 24 heures.

Affichage des notifications système

L'en-tête de Video Cloud Studio affichera une icône en forme de cloche lorsque des notifications système sont disponibles. Un clic sur cette icône permet d'afficher la liste des notifications système.

L'en-tête de Video Cloud Studio affichera une icône d'avertissement quand il y a des problèmes de système. Cliquez sur l'icône pour ouvrir la page État du système Brightcove.

Gestion des paramètres

Le menu des paramètres de Video Cloud Studio est accessible depuis le ADMIN menu. Le menu Admin de Video Cloud permet d'accéder à :

  • Informations sur le compte - Afficher des informations sur le compte Video Cloud actuel
  • Gérer les utilisateurs - Gérez les utilisateurs autorisés à utiliser Video Cloud Studio
  • Paramètres du compte - Configurer les paramètres généraux du compte
  • Paramètres de téléchargement: offre des options de personnalisation du module Upload
  • Profils d'ingestion - Gérez les profils d'ingestion dans votre compte Video Cloud
  • Champs vidéo - Marquer les champs vidéo standard comme requis et afficher, créer et modifier des champs de métadonnées personnalisés pour le compte
  • Paramètres de langue - Configurer la prise en charge d'autres langages de métadonnées
  • Droits de lecture - Configurer les droits de lecture
  • Partage multimédia - Partager des vidéos sur d'autres comptes
  • Paramètres d'annonce côté serveur - Configurer les paramètres d'annonce côté serveur
  • Paramètres du lecteur: permet d'utiliser l'authentification de base du nom d'utilisateur et du mot de passe pour accéder aux API de gestion du lecteur et du système de mise à disposition
  • Intégrations - Intégrez les réunions Zoom dans Brightcove Live
  • Live Settings - Gérez les paramètres utilisés par le module Live
  • Authentification API - Création et gestion des informations d'identification d'API
  • Facturation - Afficher les informations de facturation du compte
Note: Vous devez être administrateur de compte pour avoir accès au menu ADMIN.

Sélection d'un compte

Si vous avez accès à plusieurs comptes Video Cloud, la liste déroulante de sélection de compte Video Cloud permet de sélectionner le compte Video Cloud auquel vous souhaitez accéder. Pour changer de compte, cliquez sur votre nom - compte dans le coin supérieur droit de Video Cloud Studio. Le compte actuel est indiqué par un point placé en regard du nom de compte.

Cliquez sur le lien Mon profil pour mettre à jour vos informations d'utilisateur et votre mot de passe. Pour plus d'informations, reportez-vous à la section Gestion des informations de profil.

Accès aux modules

Le menu Video Cloud Studio permet d'accéder aux modules Studio. Accédez au menu en cliquant DOMICILE.

Notez que les modules qui s'affichent sont basés sur votre abonnement Video Cloud et votre profil utilisateur.

  • Charger - Charger des vidéos sur votre compte Video Cloud
  • Media - Définir les propriétés vidéo, organiser les vidéos à l'aide de dossiers et publier des vidéos
  • Analytics - Afficher des analyses sur les performances vidéo et du lecteur
  • Lecteurs - Créez et stylisez des lecteurs et générez du code de publication pour les lectuers
  • Live - Créez et diffusez des événements en direct
  • Cloud Playout - Créez des chaînes linéaires en direct pour les vidéos
  • Galerie - Créez des portails vidéo pour votre contenu vidéo
  • Social - Distribution de vidéos sur des comptes de médias sociaux
  • Campagne - Connecte l'analyse vidéo de Video Cloud aux plateformes d'automatisation du marketing

Affichage du tableau de bord

La page d'accueil de Video Cloud Studio affiche le tableau de bord Video Cloud. Le tableau de bord contient une série de « cartes » qui fournissent des statistiques de haut niveau sur les vidéos de votre compte. Les cartes suivantes sont disponibles :

  • Vues vidéo - Affiche le nombre de vues vidéo par jour pendant la période sélectionnée. En cliquant sur le Voir les détails de la vidéo Le lien ouvrira le rapport sur les performances où vous pourrez voir les performances vidéo
  • Résumé du compte: affiche le nombre total de vidéos et de lectuers dans le compte et le nombre de vidéos récemment créées et mises à jour. Cliquez sur un numéro pour ouvrir le module Media ou Lecteurs.
  • Type de périphérique: affiche le type d'appareils utilisés pour regarder des vidéos pendant la période sélectionnée. En cliquant sur le Voir les vues vidéo par type d'appareil Le lien ouvrira le rapport sur les performances où vous pourrez afficher les performances vidéo par type d'appareil.
  • Quoi de neuf ?- Affiche les mises à jour des produits Video Cloud. Pour une liste complète des mises à jour, consultez le Notes de mise à jour de Video Cloud Studio.
  • Vidéos les plus tendances - Affiche les vidéos ayant le plus de vues pour la période sélectionnée. Cliquez sur le lien du titre de la carte pour ouvrir le rapport de performance dans lequel vous pouvez visualiser les performances vidéo.
  • Voir les régions: affiche les emplacements des visionneuses pour la période sélectionnée. En cliquant Voir les visionnages vidéo par pays ouvrira le rapport sur les performances dans lequel vous pourrez voir les vues vidéo par pays.

Localisation

La langue de Video Cloud Studio peut être modifiée en changeant la langue de votre profil. Les langues suivantes sont prises en charge :

  • Anglais
  • Espagnol (Espagne)
  • Japonais
  • Français (France)
  • Chinois (traditionnel)
  • Coréen

Pour plus d'informations, reportez-vous à la section Gestion des informations de profil.
