A

Médias actifs
Nombre de vidéos actives dans le compte. Notez que ces données ne sont pas en temps réel: elles sont synchronisées à partir de votre vidéothèque une fois par jour. Notez également que ces données peuvent ne pas être exactes si vous regardez des données analytiques « en temps réel » (datant de moins de 3 jours).
API Analytics (Statistiques)
Interface de programmation d'applications basée sur REST qui permet aux développeurs de récupérer des mesures d'analyse pour les vidéos Video Cloud directement au format JSON. Les API d'analyse fournit les données pour l'interface utilisateur d'analyse dans Video Cloud Studio.
Clé d'accès
Clé OAuth2 temporaire permettant d'accéder aux API REST, telles que l'API CMS, l'API Analytics et l'API Dynamic Ingest.
Associé
Éditeurs internet qui assurent la syndication des lecteurs Video Cloud.
Demande d'annonce
Requête (appel) émise par un lecteur Video Cloud vers un serveur publicitaire. Il s’agit d’une requête HTTP. Elle inclut les combinaisons clé/valeur contenant des informations qui permettront au serveur de choisir une publicité appropriée au contexte. Une requête publicitaire ne signifie pas le renvoi systématique d'une publicité.
API d'analyse
Les API d'analyse vous permet d'obtenir par programmation des données d'analyse pour les comptes et les vidéos Video Cloud.
Apple HLS
Apple HTTP Live Streaming, un protocole implémenté par Apple pour fournir des vidéos à la demande. Apple HLS divise la vidéo en une séquence de petits téléchargements de fichiers, chaque téléchargement chargeant une petite partie de la vidéo via HTTP. La diffusion HTTP Apple en direct n'a rien à voir avec la diffusion vidéo en direct.
Rapport des dimensions
Rapport hauteur/largeur des images d'une vidéo. 4:3 pour les systèmes standard et 16:9 pour HDTV.

B

Bande passante
Plage des fréquences de signal qu'un équipement audio ou vidéo peut coder ou décoder. La vidéo utilise une fréquence plus élevée que l'audio et requiert donc davantage de bande passante.
Blog
Publication sur l'internet impliquant des ajouts périodiques classés par ordre antichronologique.
Lecteur Brightcove
Le lecteur Brightcove est un lecteur hautes performances, multiplates-formes prioritairement HTML5. Il se charge rapidement, diffuse des vidéos en HLS à débit adaptatif de haute qualité les ordinateurs de bureau et les dispositifs mobiles et s'intègre à vos systèmes publicitaires et statistiques favoris.
API du lecteur Brightcove
L'API du lecteur Brightcove peut être utilisée pour contrôler automatiquement l'aspect et le fonctionnement du lecteur Brightcove.
Chaîne à haut débit
Ensemble de vidéos connexes distribuées par internet.
Intercalaire
Courte séquence vidéo s’affichant avant une vidéo plus longue ou au chargement d’un lecteur.

C

API CMS
Les API CMS fournit un accès en lecture et en écriture non mis en cache aux métadonnées de la vidéo.
Codec
Algorithme utilisé pour capturer le contenu vidéo ou audio analogique sous forme numérique. Comparer avec récipient.
Conteneur
Le conteneur vidéo est un format qui définit la manière dont les métadonnées sont stockées avec les données audio et vidéo. Il ne définit pas l’encodage ou la compression des données. Par exemple, MP4 peut être un conteneur pour les fichiers H.264.
Réseau de diffusion de contenu (CDN)
Un service de serveurs en réseau qui diffusent de façon transparente le contenu (en particulier le contenu de grande taille) aux utilisateurs finaux. Les deux CDN principaux sont Akamai et Limelight.
Ciblage de contenu
Le ciblage de contenu est un processus utilisé pour afficher des contenus différents aux visiteurs en fonction des informations connues sur ces visiteurs.
CSS
Les feuilles de style en cascade constituent des mécanismes de contrôle standard de la conception d’une page HTML. CSS est utilisé pour définir les lecteurs Brightcove.
Marqueurs temporels
Des marqueurs invisibles insérés à des endroits spécifiques d’une vidéo. Ils peuvent servir à déclencher des événements externes tels qu'une requête publicitaire mid-roll ou un sous-tirage contextuel. Vous pouvez définir des points de repère à l'aide du module Media.

D

DASH (MPEG-DASH)
Le Dynamic Adaptive Streaming over HTTP (DASH), plus connu sous le nom de MPEG-DASH, est une technologie de diffusion adaptative qui permet de diffuser un contenu de haute qualité sur internet à partir de serveurs internet HTTP classiques. Cette technologie (équivalente au Apple's HTTP Live Streaming (HLS)), divise le contenu en une séquence de petits segments de fichier HTTP.
Restriction du domaine
Forme basique de sécurité de contenu qui empêche un lecteur de fonctionner, excepté dans les domaines spécifiés.
Gestion des droits numériques (DRM)
Digital Rights Management (DRM) est une technique de cryptage et de compression des fichiers médias de sorte qui ne permet leur décryptage qu'à l'aide d'une clé.
Livraison dynamique
Voir Diffusion en continu à plusieurs débits.
API Dynamic Ingest
L'API Dynamic Ingest (DI) est utilisée pour demander à Video Cloud de récupérer une ressource vidéo et de créer un rendu pour celle-ci. Les API d'ingestion dynamique peut également être utilisé pour ingérer des images miniatures et des affiches ou des fichiers WebVTT pour les légendes, les chapitres ou les sous-titres.

E

Encodage
Conversion des données audio et vidéo en un flux pouvant être lu dans un lecteur.
Cryptage
Processus d'encodage des vidéos permettant aux seules parties concernées de les visionner.

F

Frame Rate (Nombre d’images par seconde)
Nombre d'images par seconde affichées lors de la lecture d'un clip vidéo.

G

Géo-filtrage
Imposition de restrictions géographiques aux vidéos, généralement en vue de protéger les droits de propriété.

H

Format H.264
Également appelé MPEG-4 AVC (Advanced Video Coding). Format de fichier utilisé pour diffuser la vidéo sur internet. Le format H.264 est conseillé pour la diffusion de vidéos en ligne. Les fichiers H.264 utilisent généralement le conteneur MP4 et possèdent une extension de fichier .mp4. Ce format fournit un niveau de compression supérieur à celui des formats précédents.
HLS
HTTP Live Streaming: format vidéo créé par Apple qui rend une vidéo disponible en segments courts diffusés côté client.
HTML
(HyperText Markup Language) Langage conçu pour la création des pages internet avec données hypertexte et autres devant être affichées dans un navigateur.
HTML5
HTML5 est un langage de balisage internet utilisé pour structurer et présenter le contenu destiné au World Wide Web. HTML5 est la cinquième révision du langage HTML standard du World Wide Web Consortium (W3C).

I

Profil d'intégration
Les profils d'intégration contiennent les paramètres de conversion de code qui seront utilisés lors du chargement des vidéos à l'aide du module de chargement ou de l'API Dynamic Ingest.
API des profils d'intégration
L'API des profils d'intégration vous permet de créer, récupérer, mettre à jour et supprimer les profils d'intégration pour votre compte Video Cloud. Les profils d'intégration définissent les rendus créés lorsque vous ajoutez une vidéo dans Video Cloud.

J

JavaScript
Un langage de script mieux connu pour son utilisation dans les sites Web, bien qu'il soit également utilisé pour permettre l'accès par script aux objets incorporés dans d'autres applications.
JPEG (Joint Photographic Experts Group)
Format de compression utilisé pour les images numériques. Le format JPEG est le format d’image standard utilisé par le service Brightcove pour les vignettes, les captures d’écran et les images d’arrière-plan.

K

Combinaisons clé/valeur
Lorsqu’un lecteur envoie des requêtes publicitaires, il y inclut une ou plusieurs combinaisons clé/valeur (par exemple, category=sports) qui donnent au serveur les informations nécessaires sur le contexte de la publicité et améliore ainsi le ciblage.

L

Letterbox
Méthode d'affichage des vidéos en écran large sur un écran de format différent, qui consiste à laisser un espace, généralement des barres noires, au-dessus et sous l'image.
Logo
Image réservée représentant de manière symbolique la marque d’une entreprise.
Logo superposé
Petite image transparente affichée dans la fenêtre de diffusion de la vidéo. Parfois aussi appelée filigrane.
Boucle
Lorsque les vidéos d’une sélection ont toutes été diffusées et que la lecture recommence à la première vidéo.
Vidéo en streaming en direct
Un protocole de diffusion de vidéo d'un événement sur le vif, en transmettant les flux enregistrés et capturés directement, à l'inverse de la distribution de fichiers encodés pour la diffusion comme la VOD.

M

Media Library
Collection de vidéos, de listes de lecture et d'autres ressources dans votre compte Video Cloud.
Métadonnées
Les métadonnées se rapportent aux descriptions, aux balises, au genre et à d’autres données introduites dans la définition de la vidéo.
Système d’évaluation de film MPAA
Un système institué par la Motion Picture Association of America destiné à évaluer un film en fonction de son contenu. Le système d’évaluation MPAA (G, PG, R, etc.) est le système le plus réputé en ce qui concerne la définition de contenu potentiellement choquant. Son utilisation est généralement limitée au secteur cinématographique, car MPAA dispose de marques déposées pour chaque évaluation.
Diffusion en continu à plusieurs débits
Une fonctionnalité qui prend en charge plusieurs versions, ou restitutions, pour une vidéo unique et permet de fournir un rendu qui correspond le mieux à la taille du navigateur et à la bande passante du spectateur. Nous appelions cela « diffusion dynamique » dans les versions précédentes.
Multiplexage/démultiplexage (en anglais « muxing / demuxing »)
Écriture et lecture de flux audio et vidéo vers et depuis divers récipient formats.

N

SDK du lecteur natif Brightcove
Les SDK de lecteur natifs Brightcove fournissent les outils nécessaires pour créer des applications vidéo pour iOS et Android.

O

OAuth
OAuth est une norme ouverte d'autorisation. OAuth fournit aux applications clientes un accès délégué sécurisé aux ressources du serveur' pour le compte d'un propriétaire de ressources.
API OAuth
L'API OAuth est une API RESTful qui vous permet d'obtenir des clés d'autorisation pour accéder aux autres API REST de Brightcove.

P

Lecture
Lecture d’un fichier vidéo dans un lecteur Video Cloud.
API de lecture
L'API de lecture peut être utilisée lors de l'exécution pour accéder aux données vidéo et des sélections d'applications mobiles ou de portails vidéo.
API de gestion des lecteurs

L'API de gestion des lecteurs vous permet de :

  • Créer un lectuer
  • Afficher les paramètres d'un lecteur
  • Modifier les paramètres d'un lecteur
  • Afficher la liste de tous les lecteurs associés à un éditeur
  • Publier un lecteur pour le mettre à la disposition des utilisateurs
Code de publication du lecteur
Le code « extrait » utilisé pour incorporer un lecteur Brightcove dans une page Web.
Sélection
Ensemble de vidéos groupées dans un ordre particulier destiné à être lu dans un lecteur Video Cloud.
API Playlist
L'API Playlist est utilisée pour contrôler automatiquement une sélection en utilisant JavaScript.
API Policy
L'API Policy est utilisée pour créer ou obtenir des clés qui peuvent être utilisée pour accéder à l'API de lecture. Les clés sont générées automatiquement pour les lecteurs Brightcove.
Image de l'affiche
Image fixe affichée dans la zone de lecture d’un lecteur vidéo avant ou après la lecture de la vidéo.
px
Abréviation de « pixels » utilisée dans CSS.

R

Rendu
Lorsque vous utilisez la diffusion en continu à plusieurs débits, vous pouvez créer plusieurs versions de la même vidéo, chacune étant associée à un débit et à une taille distincts. Chaque version devient ainsi un rendu. La diffusion en continu à plusieurs débits permet de sélectionner le meilleur rendu en fonction de la taille du lecteur et de la vitesse de connexion de l’utilisateur.
ID de référence
Identifiant unique affecté par un éditeur à une vidéo ou une sélection. L’ID de référence (ID ref) doit être distinguée de l’ID de la vidéo ou de la sélection qui sont des identifiants uniques générés par Video Cloud. L’ID de référence permet la coordination de Video Cloud et d’autres services, comme le système de gestion de contenu de l’éditeur.
Flux RSS
La syndication est une technologie particulièrement populaire. Elle permet de signaler aux utilisateurs les mises à jour du contenu d’un site internet, d’un blog ou d’une chaîne TV sur internet.

S

Moteur de recherche
Programme (Google par exemple) permettant de trouver des informations sur l'internet.
Site Web du Réseau Social
Site internet (Facebook ou Twitter, par exemple) permettant à ses membres de partager des messages, des éléments multimédias et d’autres informations entre eux.
Streaming
Format d'encodage à faible débit destiné à une utilisation sur les réseaux et sur internet. La diffusion en continu fournit de la vidéo sous forme de petits segments avec un encodage dont le débit est adapté à la vitesse de connexion de l'utilisateur, de sorte que l'utilisateur à distance peut lire les contenus audio ou vidéo avec un nombre limité d'interruptions sans avoir à télécharger au préalable l'intégralité du fichier vidéo.

T

Balises
Mots-clés ou phrases isolés permettant l’organisation du contenu. Brightcove autorise l‘utilisation de balises pour que les éditeurs puissent organiser, classer les vidéos et pour permettre la création de sélections.
Vignette
Petite image associée à une vidéo dans les listes et permettant d’identifier la vidéo dans les lecteurs, les sites internet et les moteurs de recherche.
Authentification basée sur des jetons
Les systèmes d'authentification à partir de jetons permettent aux utilisateurs de saisir leur nom d'utilisateur et leur mot de passe pour obtenir une clé les autorisant à récupérer une ressource spécifique sans nécessiter leur nom d'utilisateur et leur mot de passe. Après avoir obtenu sa clé, l'utilisateur peut la donner et ainsi donner accès à une ressource spécifique pour une période donnée.
Contrôle parental
Système d’évaluation du contenu établi par la Federal Communications Commission (FCC) américaine destiné à classer les émissions TV en fonction du public auquel elles sont destinées sur la base du contenu.

U

URL
Adresse permettant d’identifier une ressource et sa position sur internet (par exemple, http://www.brightcove.com).

V

Débit variable
Schéma de compression suivant lequel chaque unité de contenu d'entrée peut être compressée selon des tailles différentes. Pour la vidéo MPEG-2, par exemple, cela signifie que les séquences « plus faciles » (c'est-à-dire sans mouvement) peuvent se compresser à de très petites tailles, tandis que les séquences « dures » (avec beaucoup de mouvements et de découpes de scène) peuvent se compresser à des tailles beaucoup plus grandes.
Vidéo

Titre vidéo unique à diffuser dans le lecteur Video Cloud. Une vidéo est basée sur un fichier vidéo et peut se présenter sous différents formats. Elle est ensuite convertie au format H.264 ou FLV, si nécessaire. Par ailleurs, la vidéo peut aussi inclure des métadonnées telles que :

  • name
  • courte description
  • longue description
  • tags
  • capture d’écran
  • image de garde
  • intercalaire
  • logo superposé
Video Cloud Studio

Application internet permettant l’accès et le pilotage du service Video Cloud. Studio comprend une série de modules permettant d'exécuter différentes tâches :

  • Charger - Charger des vidéos sur votre compte Video Cloud
  • Media - Définir les propriétés vidéo, organiser les vidéos à l'aide de dossiers et publier des vidéos
  • Lecteurs - Créez et stylisez des lecteurs et générez du code de publication pour les lectuers
  • Analytics - Afficher des analyses sur les performances vidéo et du lecteur
  • Live - Créez et diffusez des événements en direct
  • Galerie - Créez des portails vidéo pour votre contenu vidéo
  • Audience - Connecte l'analyse vidéo de Video Cloud aux plateformes d'automatisation du marketing
  • Social - Distribuez des vidéos sur YouTube, Facebook et Twitter
Outils de montage vidéo
Application logicielle pour l’édition des fichiers vidéo sur un ordinateur.
Image de l'affiche vidéo
Image fixe affichée dans la zone de lecture d’un lecteur vidéo avant ou après la lecture de la vidéo.
VOD
Vidéo à la demande. Un protocole qui permet la lecture de fichiers vidéo distribués par streaming ou téléchargement progressif à volonté, c'est-à-dire à la demande. Contrairement à vidéo en direct.

W

WebVTT
Format Web Video Text Tracks. Les fichiers WebVTT fournissent les titres ou les sous-titres de contenu vidéo, ainsi que les descriptions de vidéos, les chapitres de navigation et, plus généralement, toute forme de métadonnées synchronisées avec le contenu audio ou vidéo. Les lecteurs Brightcove utilisent des titres au format WebVTT.

X

XLSX
Fichier Microsoft Excel. Vous pouvez exporter les données dans ce format.
