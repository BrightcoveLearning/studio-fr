Génération du code intégré pour un canal Cloud Playout

Pour générer le code d'intégration d'un canal Cloud Playout, vous utiliserez le module Live. Pour plus de détails sur l'utilisation du module Live, consultez Création et gestion d'événements en direct à l'aide du module en direct.

Deux types de code intégré sont disponibles :

  • Standard - code d'intégration iframe. Recommandé dans la plupart des cas. Ce type place le lecteur à l'intérieur d'une balise <iframe>.
  • Avancé - Code intégré dans la page. Permet au lecteur d'exister directement dans la page HTML, pas dans un iframe. Cela permet aux développeurs d'accéder facilement au lecteur et aux propriétés et événements associés.

Vous avez également la possibilité d'obtenir l'URL vers le lecteur publié. Ceci est utile pour afficher l'événement dans un navigateur.

Pour générer le code d'intégration d'un événement en direct, procédez comme suit :

  1. Ouvrez le module Live.
  2. Localisez l'événement en direct sur lequel la chaîne sera diffusée en continu. En règle générale, l'événement aura lieu dans le Etre prêt onglet et avoir un nom de Canal sauf si vous avez sélectionné un événement spécifique vers lequel diffuser lorsque la chaîne a été créée.
  3. Cliquez sur le nom de l'événement pour ouvrir le Salle de contrôle page.
  4. Cliquez sur Publier et incorporer> Configuration du lecteur dans la navigation de gauche.
    Note: Si le canal est configuré pour activer les publicités côté serveur (SSAI), Brightcove Player version 6.18.3 et supérieure doit être utilisé. Seuls les lectuers de la version 6.18.3 et supérieure apparaîtront dans la liste déroulante.
  5. Cliquez sur l' URL du lecteur pour prévisualiser l'événement dans un navigateur ou copier le code d'intégration approprié à utiliser.

Si le programme a été créé avec Insertion d'annonces option définie sur Au , les options de publication incluront un Sélectionnez la configuration de l'annonce liste déroulante où la configuration publicitaire à utiliser peut être sélectionnée.
