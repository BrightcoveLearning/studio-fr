Un marqueur temporel sert à indiquer un moment précis dans une vidéo. Vous pouvez les utiliser pour déclencher des spots publicitaires mid-roll (des publicités qui s'affichent pendant la lecture de la vidéo, comme une page de publicité) ou pour séparer des chapitres ou des scènes dans une vidéo de longue durée.

Il existe deux types de points de repère, mais les deux fournissent simplement une valeur de chaîne au type propriété de l'objet point de repère. La valeur appropriée n'est utile que si elle est utilisée dans JavaScript personnalisé lors du traitement des informations de point de repère. 

  • Un d - Attribue la valeur de chaîne de UN D à la taper biens
  • Code - Attribue la valeur de chaîne de CODE à la taper biens

Des points de repère peuvent être ajoutés aux vidéos à l'aide du module Media ou à l'aide de l'API Catalogue. Pour plus d'informations sur l'ajout de points de repère à l'aide de l'API de catalogue, consultez le document du développeur Implémentation des points de repère par programme.

Note: Les points de repère ne peuvent pas être ajoutés aux vidéos de ressources distantes en direct.

Ajout de points de repère

Pour ajouter des points de repère à une vidéo, ouvrez le module Media et procédez comme suit :

  1. Cliquez sur le lien du titre de la vidéo pour ouvrir la page des propriétés de la vidéo.
  2. Utilisez le lecteur de prévisualisation pour lire la vidéo. Mettez la vidéo en pause à l'endroit où vous souhaitez ajouter un point de repère.
  3. Cliquez sur Ajouter un point de repère.
  4. Si nécessaire, ajustez le Code temporel (hh:mm:ss) valeurs pour le point de repère. Notez que le secondes Le champ prend en charge les millisecondes après la virgule décimale.
  5. Sélectionnez le Type du point de repère, puis ajoutez les métadonnées appropriées.
    • ad - Une demande d'annonce sera faite. Lorsque vous utilisez des points de repère publicitaire, le lecteur doit être configuré pour la publicité.
    • code - Les points de repère seront gérés par programmation. Voir le Implémentation des points de repère par programme document développeur pour plus d'informations.
  6. Entrez un Nom pour le point de repère. La valeur du nom est accessible par programmation.
  7. Entrer Métadonnées (pour les points de repère de code) ou Paires clé / valeur (pour les points ad cue).
    • Métadonnées: métadonnées facultatives stockées avec le point de repère. Les métadonnées sont accessibles par programmation. Un exemple d'utilisation d'un point de repère de code et de métadonnées de transmission est lorsque vous souhaitez afficher différents appels à l'action pendant la lecture de la vidéo. Voir l'exemple de lecteur Brightcove: CTA d'affichage des points de repère à titre d'exemple.
    • Paires clé/Valeur: paires clé/valeur pouvant être transmises à une demande d'annonce. Une paire clé-valeur est un ensemble de deux éléments de données liés: une clé, qui est un identifiant unique pour un élément de données, et la valeur, qui est soit les données identifiées, soit un pointeur vers l'emplacement de ces données. Les paires clé/valeur peuvent être utilisées pour fournir des publicités plus ciblées. Tout ce qui est entré ici sera copié dans le tag d'emplacement publicitaire où le {mediainfo.ad_keys} macro est insérée. Pour plus d'informations, consultez Publicité avec le plugin IMA3.

Après l'ajout de points de repère à une vidéo, au moment de l'exécution, les points de repère sont convertis en éléments de repère HTML5 WebVTT. Lorsque cette conversion a lieu, certaines données de point de repère, telles que le type et l'heure du point de repère, sont converties en piste de texte. La valeur temporelle du point de repère sera mappée dans le repère WebVTT startTime et endTime Propriétés. Voir ce document pour plus d'informations sur l'interface VTTCue.

Modification des points repères

Pour modifier des points de repère dans une vidéo, ouvrez le module Media et procédez comme suit :

  1. Cliquez sur le lien du titre de la vidéo pour ouvrir la page des propriétés de la vidéo.
  2. Cliquez sur le point de repère à modifier dans la chronologie.
    Note: Une fois en mode édition, le Précédent et Prochain Les boutons peuvent être utilisés pour naviguer vers d'autres points de repère.
  3. Modifiez les propriétés du point de repère, puis cliquez sur sauver.
Note: Il n'est pas possible d'avoir plusieurs points de repère avec le même code temporel.

Suppression de points repères

Pour supprimer des points de repère, ouvrez le module Media et procédez comme suit :

  1. Cliquez sur le lien du titre de la vidéo pour ouvrir la page des propriétés de la vidéo.
  2. Cliquez sur le point repère à supprimer dans la chronologie.
  3. Cliquez sur l'icône Supprimer () à côté des propriétés du point de repère.
  4. Confirmez la suppression.
