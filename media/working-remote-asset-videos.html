En règle générale, les vidéos sont chargées sur Video Cloud avec le module Upload. Ces fichiers sont alors entretenus par Brightcove. Les ressources distantes vous permettent de conserver vos fichiers vidéo localement ou sur un système tiers. Vous envoyez ensuite l’URL du fichier vidéo à Video Cloud.

Note: La possibilité de créer des vidéos d'actifs distants est disponible pour les éditeurs de Video Cloud Enterprise. Si vous souhaitez mettre à niveau votre compte Video Cloud, veuillez contacter Brightcove pour plus d'informations.

Elles peuvent également être créées avec l'API Video Cloud CMS. Pour plus d'informations, consultez la documentation des développeurs. Vidéos de ressources à distance créées à l'aide du API CMS peuvent ensuite être gérés à l'aide du module Média.

Accès aux ressources distantes

Tous vos fichiers de ressources distants doivent être disponibles par le biais d’une URL publique, sans authentification ni contrôle d'accès nécessaires. Si les ressources distantes sont hébergées par un CDN, leur URL doit correspondre à l’URL des ressources dans le CDN et non à celle du serveur d’origine.

Création d'une vidéo distante

Pour créer une vidéo distante, procédez ainsi.

  1. Ouvrez le module Médias.
  2. Cliquez sur Ajouter une vidéo à distance dans le panneau de navigation latéral. La boîte de dialogue Ajouter une vidéo à distance s'ouvre.
  3. Saisissez le nom de la vidéo. Ce nom sera affiché dans le module Médias ainsi que dans le lecteur.
  4. Sélectionnez le type de la vidéo, Live ou On Demand.
  5. Pour les types de vidéos On Demand, entrez la durée au format hh:mm:ss.
  6. Pour chaque rendu, saisissez l'URL de rendu et le format. Cliquez sur Ajouter un format associé.
    Note: Les URL relatives au protocole sont prises en charge.
  7. Cliquez sur Save pour enregistrer la ressource distante. La vidéo de l'élément distant sera répertoriée dans le module Médias une fois la page actualisée.

Les vidéos de ressources distantes ne comportent pas d'affiches ou d'images miniatures, cependant, les images peuvent être téléchargées. Pour plus d'informations sur le téléchargement d'images d'affiches et de miniatures, reportez-vous à la section Chargement d'images d'affiche et de vignettes.

Modification d'une vidéo distante

Pour modifier une vidéo distante, procédez ainsi.

  1. Ouvrez le module Médias.
  2. Cliquez sur le titre de la vidéo distante pour ouvrir ses propriétés.
  3. Localisez le Fichiers vidéo - Actifs distants rubrique et cliquez sur Éditer.
  4. Cliquez sur une ligne Source pour la sélectionner. Cliquer sur l'URL source l'ouvrira dans un autre onglet du navigateur.
  5. Mettre à jour le URL et Durée comme requis. Cliquez sur Save pour enregistrer les modifications apportées.

Contraintes

  • Vous ne pouvez pas utiliser la fonction de capture d'image pour créer des images fixes et miniatures pour les vidéos de ressources distantes, cependant, des images fixes et miniatures vidéo peuvent être téléchargées pour les vidéos de ressources distantes.
  • Vidéos de ressources distantes créées dans De base les comptes ne peuvent pas être modifiés après leur création
  • SSAI n'est pas pris en charge pour les actifs distants
  • DRM n'est pris en charge que si vous pouvez personnaliser le lecteur pour communiquer avec le serveur de licences DRM. Ceci est possible avec le Brightcove Player via un plugin. Pour les autres lectuers, vous devrez consulter la documentation du lectuer.
  • Analytics: le système Brightcove Analytics dépend des métadonnées vidéo générées par le système, telles que la durée de la vidéo. Les données d'analyse des ressources distantes seront moins riches, à moins que vous ne puissiez fournir ces métadonnées pour la ressource distante.
  • Garantie de lecture: toutes les vidéos ingérées dans Video Cloud peuvent être lues dans le Brightcove Player. Cela est vrai pour presque tous les actifs distants, mais il existe des cas de périphérie où les vidéos peuvent ne pas être lues. Il y a quelques fonctionnalités avancées de DASH et HLS que nous ne prenons pas encore en charge par exemple. Pour les lecteurs non BrightCove, vous devrez vérifier que le format vidéo est entièrement pris en charge par ce lecteur.
  • Les téléspectateurs peuvent ne pas être en mesure de lire les vidéos de ressources distantes si l'heure de leur ordinateur est réglée en avance sur l'heure réelle.
  • Les points de repère ne peuvent pas être ajoutés aux vidéos de ressources à distance en direct
