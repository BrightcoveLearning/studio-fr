Brightcove se réserve le droit de mettre à jour les profils d'ingeste par défaut associés aux comptes Video Cloud. Vous pouvez également avoir besoin de mettre à jour vos propres profils d'ingest personnalisés. Lorsque les profils d'ingest changent, vous pouvez souhaiter que les vidéos existantes de votre compte soient retranscodées afin qu'elles aient des formats associés correspondant aux nouveaux paramètres de profil d'ingest.

Note: La possibilité de retranscoder les vidéos existantes n'est disponible que pour les comptes Professionnel et Enterprise. Si cette fonctionnalité vous intéresse, veuillez contacter votre responsable de compte.

Les vidéos peuvent être retranscodées à l'aide du module Media ou à l'aide de l'API Dynamic Ingest. En utilisant le API d'ingestion dynamique peut être une meilleure solution lorsque vous avez un grand nombre de vidéos qui nécessitent un retranscodage. Pour retranscoder des vidéos à l'aide de l'API Dynamic Ingest, consultez le Présentation de l'API Dynamic Ingest document du développeur.

Note: Vous ne pouvez pas retranscoder les vidéos partagées sur votre compte.
Note: Seules les vidéos qui ont été téléchargées à l'aide du module Upload ou à l'aide de l'API Dynamic Ingest peuvent être retranscodées. Si la vidéo ne peut pas être retranscodée, le retranscodage échouera.
Remarque : les vidéos ne peuvent être retranscodées que si le master vidéo a été archivé lors de l'ajout de la vidéo au compte. Les vidéos plus anciennes peuvent ne pas avoir de maître, selon le profil d'ingestion utilisé.

Processus de reconversion d'une vidéo

Procédez comme suit pour remplacer les formats associés existants pour une vidéo.

  1. Ouvrez le module Médias.
  2. Cliquez sur la vidéo que vous souhaitez retranscoder.
  3. Dans le Fichiers vidéo rubrique, cliquez sur Retranscoder.
  4. Sélectionnez un profil d'ingest à utiliser.
  5. Vérifier Conserver les images de vignettes et d'affiche existantes pour conserver les images existantes. Si vous décochez cette case, de nouvelles images seront générées en fonction des paramètres d'image dans le profil d'ingest sélectionné.
    Notez que si vous cochez la case, les images existantes seront conservées - cependant, les URL des images vont changer.
  6. Cliquez sur Retranscode.

La vidéo reste accessible lors de la création de rendus, grâce aux rendus existants.
