La fonctionnalité Quick Edit permet de modifier simultanément et simplement les propriétés de plusieurs vidéos. Plusieurs vidéos peuvent être sélectionnées en cliquant sur une case à cocher puis en cliquant sur une autre case à cocher (maintenez la touche Maj enfoncée tout en cliquant sur la deuxième vidéo, et les deux vidéos et toutes celles entre elles seront sélectionnées). En cliquant Edition rapide ouvrira le panneau d'édition rapide sur le côté droit du module Média.

Le volet Quick Edit affiche un sous-ensemble des propriétés vidéo :

  • Name
  • Courte description
  • Long Description
  • Tags to Add
  • Tags to Remove
  • Related Link URL
  • Related Link Text
  • Advertising
  • Ad Keys
  • Scheduling Start/End Dates
  • Geofiltering
  • Champs personnalisés
  • Restrictions de lecture
Note: Pour une description de toutes ces propriétés, consultez le Modification des propriétés vidéo à l'aide du module multimédia document.

Servez-vous du volet Quick Edit pour attribuer des valeurs. Cliquez sur Save pour enregistrer les modifications apportées ou sur Cancel pour quitter.

Remarques relatives à l'utilisation de Quick Edit

  • Lorsqu'au moins une vidéo a des valeurs affectées à une propriété, Edit Mixed Value s'affiche
  • Un astérisque (*) suivant le nom de la propriété indique une propriété obligatoire
  • Pour annuler les modifications apportées à une propriété, cliquez sur Restore.
