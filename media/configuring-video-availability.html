Les propriétés de disponibilité des vidéos permettent de contrôler les dates et heures de visionnage d'une vidéo. Le géofiltrage peut être utilisé pour contrôler les pays qui ont le droit de lire la vidéo.

Remarques:
  • Le format de données utilisé pour la planification des dates utilise le format américain: MM/JJ/AAAA.
  • Les propriétés de disponibilité vidéo ne sont disponibles que pour les comptes Enterprise et Professional.
  • La planification d'une vidéo n'a aucun effet sur la question de savoir si une vidéo est ou devient active ou inactive. La planification affecte uniquement les vidéos actives.
  • Des propriétés de filtrage géographique et de planification peuvent également être ajoutées à l'aide du API CMS quand vous créer ou mettre à jour une vidéo.
  • Certains éditeurs ont demandé la possibilité de filtrer les vidéos vers les bases militaires américaines ainsi qu'aux États-Unis. Pour soutenir cela, il existe un Armée américaine option. Bien qu'il n'existe pas de moyen 100% fiable d'identifier une base militaire, lorsqu'une adresse IP qui se résout sur un domaine .mil est rencontrée et n'est pas aux États-Unis, elle sera identifiée comme provenant d'une base militaire américaine plutôt que de son pays d'origine réel. Les adresses .mil qui se trouvent aux États-Unis seront toujours identifiées comme étant « us& » « Quot » ;.

Pour afficher et modifier les propriétés de disponibilité d'une vidéo, ouvrez le module Médias, cliquez sur le lien de titre de la vidéo et localisez la section Availability.

Cliquez sur le bouton Edit ou double-cliquez sur la section pour modifier les propriétés Availability. La disponibilité des vidéos peut être configurée d'après :

  • Date et heure
  • Filtrage géographique

Configuration de la date et l'heure

Une date/heure de début et une date/heure de fin peuvent être mises en place pour les vidéos. Les dates et heures permettent de contrôler quand une vidéo sera disponible à la lecture.

Par défaut, les vidéos sont disponibles immédiatement et n'ont pas de date de fin. Pour configurer les dates et heures de début/fin :

  1. Cliquez sur le bouton radio à côté du champ de date/heure.
  2. Cliquez sur l'icône de calendrier et utilisez le contrôle de calendrier pour sélectionner une date.
  3. Cliquez sur l'icône de l'horloge () pour régler une heure.
    Note: Le fuseau horaire est basé sur le fuseau horaire défini sur votre ordinateur ou votre appareil.
  4. Pour revenir au contrôle du calendrier, cliquez sur l'icône du calendrier ().
  5. Cliquez sur Save.

Configuration du géofiltrage

Le géofiltrage peut être utilisé pour contrôler quels pays ont le droit de lire la vidéo ou pas.

Notes :
  • L'utilisation du géofiltrage est disponible pour les comptes Brightcove Enterprise et Professional moyennant des frais supplémentaires. Contactez Brightcove si vous désirez activer le géofiltrage dans votre compte.
  • Le filtrage géographique peut également être appliqué aux lectuers. Video Cloud applique les politiques de géofiltrage des lecteurs et des vidéos. Si un pays est exclu de l'une d'elles, la vidéo n'y sera pas disponible.

Pour définir les propriétés de géofiltrage d'une vidéo :

  1. Sélectionnez Autoriser l'affichage dans les pays suivants ou Empêcher l'affichage à partir des pays suivants.
  2. Saisissez le nom des pays pertinents. Au fur et à mesure que vous tapez, la liste des pays sera filtrée.
  3. Cliquez sur Save.
