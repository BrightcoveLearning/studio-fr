Beaucoup d'éditeurs utilisent les métadonnées des vidéos (noms, balises, ID de référence, champs personnalisés, etc.) comme outil essentiel de suivi et de rentabilisation de leur contenu. Pour que les métadonnées vidéo soient efficaces, vous devez vous assurer que chacun des champs de métadonnées dont vous dépendez est réellement présent. La fonctionnalité d'assurance des métadonnées vous permet de désigner les champs de métadonnées vidéo importants pour votre entreprise, puis d'exiger des utilisateurs qu'ils remplissent les champs de métadonnées requis pour les nouvelles vidéos avant que les vidéos ne deviennent actives.

Définition d'un champ de métadonnées comme obligatoire

Les champs de métadonnées vidéo obligatoires sont configurés dans le cadre des paramètres du compte Video Cloud. Les champs de métadonnées standard et personnalisés peuvent être marqués comme requis. Pour configurer les champs obligatoires :

  1. Clique le Administrateur lien dans l'en-tête Studio.
  2. Clique le Champs vidéo lien.
  3. Cliquez sur le nom du champ que vous souhaitez définir.
  4. Vérifier la Rendre ce champ obligatoire case à cocher.
  5. Cliquez sur Enregistrer le champ.

A tout moment, vous pouvez décocher la case Rendre ce champ obligatoire case à cocher pour n'importe quel champ si vous ne voulez plus que ce champ soit obligatoire. Video Cloud ne vérifiera plus si ces champs ont été remplis.

Quels champs de métadonnées peuvent être requis ?

Le champ de nom de la vidéo est toujours requis. Tout champ de métadonnées personnalisé peut être un champ requis. Les champs de métadonnées standard suivants peuvent également être requis :

  • Advertising
  • Géo-filtrage
  • Description courte
  • Description longue
  • ID de référence
  • Texte du lien connexe
  • URL du lien connexe
  • Date de début planifiée
  • Date de fin planifiée
  • Balises

Vidéos sans métadonnées requises

Si une vidéo de votre médiathèque ne comporte pas toutes les métadonnées requises, alors :

  • Si la vidéo est déjà active, elle restera active et lisible. Lorsque vous spécifierez les champs de métadonnées requis pour votre compte, votre médiathèque existante ne sera pas bloquée.
  • S'il est déjà actif et que vous le modifiez dans le module Media, vous ne pouvez pas enregistrer vos modifications sans fournir toutes les métadonnées requises dans la section que vous modifiez.
  • Si elle est inactive, vous ne pouvez pas la rendre active sans saisir toutes les métadonnées requises.

Téléchargement de vidéos sans métadonnées requises

Par défaut, lorsqu'une vidéo est téléchargée à l'aide du module de téléchargement, l'état de la vidéo est défini sur actif sauf si des champs obligatoires sont manquants. Si les champs obligatoires sont manquants, un message s'affiche avec les champs obligatoires en surbrillance en rouge. Les vidéos avec des champs obligatoires manquants peuvent toujours être téléchargées, mais elles resteront Inactif jusqu'à ce que tous les champs obligatoires aient des valeurs.

Modification des vidéos

Si une vidéo est inactive, vous pouvez la modifier librement et sauvegarder ses métadonnées dans le module Médias sans saisir toutes les métadonnées requises. De plus, la vidéo ne peut pas être activée tant que toutes les métadonnées requises n'ont pas été saisies.

Si la vidéo est déjà active, vous ne pouvez pas enregistrer de modifications dans la section que vous modifiez sauf si vous fournissez toutes les métadonnées requises dans cette section ou désactivez la vidéo.

Activation des vidéos

Si vous voulez activer une vidéo dans le module Médias, elle devra comporter toutes les métadonnées requises. Si vous essayez d'activer une vidéo avec des métadonnées requises manquantes, une erreur s'affiche.

La barre d'état au bas de l'écran vous indique si vos vidéos peuvent être activées. Le bouton Activer reste désactivé jusqu'à ce que vous saisissiez toutes les métadonnées requises pour les vidéos.

Astuce concernant le processus

Vous voulez retrouver toutes vos vidéos inactives ? Dans le module Médias, vous pouvez trier toutes vos vidéos par Statut. Cliquez simplement dans l'en-tête sur le Statut colonne. Vous pourrez alors modifier chacune des vidéos inactives pour savoir les métadonnées requises qui peuvent manquer.

Contraintes

Les champs obligatoires sont appliqués uniquement dans le module Media. Vous ne pouvez pas activer une vidéo inactive dans le module Media à moins que tous les champs de métadonnées requis ne soient remplis. Toutefois, une vidéo sans métadonnées requises peut toujours devenir active des façons suivantes :

  • La vidéo a été créée dans votre compte et activée avant de spécifier les champs requis pour votre compte
  • La vidéo a été téléchargée à l'aide de l'API Dynamic Ingest

En outre, Video Cloud valide le type mais pas les valeurs des métadonnées requises. Tant que la valeur fournie a le type correct indiqué pour le champ dans les paramètres du compte (par exemple, texte, URL ou date), les utilisateurs peuvent entrer les valeurs qu'ils choisissent.
