Vous pouvez créer des dossiers dans le module Médias pour organiser vos vidéos. Vous pouvez choisir d'organiser des vidéos par sujet, auteur, produit, région, etc. Avoir des vidéos dans des dossiers les rend plus faciles à gérer dans un compte contenant des centaines ou des milliers de vidéos. Les dossiers sont affichés dans le panneau de navigation latéral. Le nombre de vidéos dans chaque dossier est affiché en regard du nom du dossier. Cliquez sur le bouton de navigation latéral pour masquer ou afficher le panneau de navigation latéral.

Note: Les dossiers imbriqués ne sont pas pris en charge.
Note: Un maximum de 100 vidéos peuvent être déplacées vers un dossier en même temps.

Création de dossiers

Pour créer un nouveau dossier, procédez comme suit :

  1. Ouvrez le module Médias.
  2. Cliquez sur Créer le dossier dans le panneau de navigation latéral.
  3. Entrez un nom pour le dossier en suivant les instructions suivantes. Appuyez sur Entrée.
    • Le nom est limité à 100 caractères
    • Les noms de dossier doivent être uniques dans un compte
    • Caractère illégal :/
Note: Un maximum de 1000 dossiers peuvent être créés.

Pour supprimer rapidement le dossier, cliquez sur le X après le nom du dossier.

Ajout de vidéos à des dossiers

Il existe plusieurs façons d'ajouter des vidéos à un dossier. Vous pouvez :

  • Ajouter plusieurs vidéos à un dossier
  • Modifiez les propriétés de la vidéo pour attribuer un dossier
Note: Il n'y a pas de limite au nombre de vidéos pouvant se trouver dans un dossier.

Ajout de plusieurs vidéos à un dossier

Plusieurs vidéos à la fois peuvent être ajoutées à un dossier en procédant comme suit.

  1. Dans le Médias module, cliquez sur Toutes les vidéos dans le panneau de navigation latéral pour afficher toutes les vidéos.
    Note: Le numéro situé après le nom du dossier indique le nombre de vidéos dans ce dossier.
  2. Sélectionnez les vidéos à ajouter à un dossier en cochant la case située en regard de la miniature de la vidéo. Cliquez sur Plus > Déplacer vers le dossier.
  3. Sélectionnez un dossier et cliquez sur Bouge toi.

Modification des propriétés vidéo pour assigner un dossier

Les vidéos peuvent être ajoutées à un dossier en modifiant les propriétés de la vidéo et en affectant un dossier.

  1. Dans le Médias module, cliquez sur le lien de la vidéo.
  2. Dans le Catégorisation rubrique, sélectionnez un Dossier. Vous pouvez également créer un nouveau dossier en cliquant sur + Créer un dossier.

Affichage de vidéos dans un dossier

Pour afficher les vidéos dans un dossier, cliquez sur un nom de dossier.

Suppression de vidéos des dossiers

Vous pouvez supprimer plusieurs vidéos d'un dossier ou supprimez une seule vidéo d'un dossier en modification des propriétés vidéo.

Suppression de plusieurs vidéos d'un dossier

Plusieurs vidéos peuvent être supprimées d'un dossier en procédant comme suit.

  1. Dans le Médias module, cliquez sur le lien du dossier.
  2. Cochez les cases à côté des vidéos à supprimer et cliquez sur Supprimer des vidéos> Supprimer des vidéos du dossier.

Modification des propriétés de la vidéo

Les vidéos peuvent être supprimées d'un dossier en modifiant les propriétés vidéo et en supprimant le dossier dans le panneau Informations vidéo.

  1. Dans le Médias module, cliquez sur le lien de la vidéo.
  2. Dans le Catégorisation section, set le Dossier à ---.

Notez que cette méthode peut également être utilisée pour sélectionner un autre dossier vers lequel déplacer la vidéo.

Suppression et changement de nom de dossiers

Les dossiers peuvent être renommés ou supprimés en procédant comme suit :

  1. Cliquez sur le dossier à renommer ou à supprimer. Cliquez sur la flèche vers le bas à côté du dossier et cliquez sur Renommer ou Effacer.
