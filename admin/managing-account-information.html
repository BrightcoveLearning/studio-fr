La page Informations sur le compte gère le nom du compte et l'adresse postale et peut être utilisée pour cloner des comptes. Notez que pour accéder à la page Administration du compte, vous devez être un utilisateur doté du rôle Administrateur.

Pour accéder à la page Administration du compte :

  1. Connectez-vous à Studio.
  2. Dans l'en-tête de navigation, cliquez sur ADMIN et alors Information sur le compte.

Les informations sur le compte seront affichées.

Pour modifier les informations de votre compte, cliquez sur Modifier les informations du compte.

Modifiez les valeurs appropriées, puis cliquez sur Enregistrer. Notez que l'ID de compte ne peut pas être modifié.

Clonage de compte

Si votre compte est activé pour le clonage de compte, un Créer un nouveau compte Le bouton apparaîtra. La fonctionnalité de clonage de compte peut être utilisée pour créer de nouveaux comptes Video Cloud. Lorsqu'un nouveau compte est créé, le support Brightcove en sera informé, puis collaborera avec votre gestionnaire de compte pour vérifier le nouveau compte (en fonction de votre contrat). Une fois vérifié, le compte sera configuré avec les fonctionnalités décrites dans votre contrat. Lorsque le compte est entièrement configuré, vous en serez averti. Pendant que la configuration est en cours, de nouveaux utilisateurs peuvent être ajoutés au compte.

Note: Le clonage de comptes n'est disponible que pour les administrateurs de comptes disposant de comptes d'entreprise.

Pour cloner le compte, cliquez sur Créer un nouveau compte, entrez un nom de compte, puis cliquez sur Enregistrer.
