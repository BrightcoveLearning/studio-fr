Les configurations d'annonces définissent divers aspects de la lecture des annonces, notamment les appels publicitaires, le timing, les balises et d'autres options de configuration des annonces. Les paramètres de publicité côté serveur définissent les annonces qui peuvent être demandées lors de vidéos vidéo à la demande (VOD) et lors d'événements en direct. Les annonces côté serveur tirent parti de la technologie SSAI (Server-Side Add Insertion) de Brightcove et offrent les principales fonctionnalités suivantes :

  • Étant donné que les annonces sont assemblées dans le flux vidéo à la demande ou en direct côté serveur, elles ne sont pas affectées par les bloqueurs d'annonces
  • Pour les événements en direct, les annonces sont insérées aux points de repère envoyés par votre encodeur ou vous pouvez créer un point de repère instantané à l'aide du module Live Control Room
  • Vous pouvez configurer des ardoises de remplissage pour remplir toute heure publicitaire inutilisée

Pour en savoir plus sur la publicité côté serveur, consultez Implémentation des annonces côté serveur avec le lecteur Brightcove.

Note :
  • Avant de pouvoir utiliser SSAI, votre compte Video Cloud doit être configuré pour la livraison dynamique et activé pour SSAI. Contactez votre représentant pour plus d’informations.
  • VMAP pour Brightcove Live SSAI n'est actuellement pas pris en charge.
  • Bien qu'il n'existe pas de fonctionnalité d'extraction préalable pour les annonces SSAI, nous déléguons la logique de position des annonces au fournisseur d'annonces. Dynamic Delivery fait tous les appels d'ajout à l'avance. Si nous recevons une annonce du fournisseur d'annonces et qu'elle est transcodée dans notre système, nous l'insérerons dans le flux.

La page Paramètres des annonces côté serveur permet de gérer les configurations des annonces en direct et VOD. Notez que pour accéder à la page Paramètres des annonces côté serveur, vous devez être un utilisateur doté du rôle Administrateur.

Pour accéder à la page Paramètres des annonces côté serveur :

  1. Connectez-vous à Studio.
  2. Dans l'en-tête de navigation, cliquez sur ADMIN et alors Paramètres d'annonce côté serveur. Une liste de toutes les configurations d'annonces s'affiche.

Pour créer une nouvelle configuration publicitaire, cliquez sur Créer une configuration publicitaire, puis cliquez sur le type d'annonce spécifique à créer :

  • Vidéo à la demande (VOD)
  • En direct

Création d'une configuration d'annonce vidéo à la demande

Note: Les configurations d'annonces côté serveur peuvent également être gérées à l'aide de l'API d'insertion d'annonces côté serveur. Pour plus d'informations, consultez le API de configuration d'annonces Video Cloud SSAI sujet.

Pour ajouter une configuration d'annonce vidéo à la demande (VOD), procédez comme suit.

  1. Sur la page Paramètres publicitaires côté serveur, cliquez sur Créer une configuration publicitaire > VOD.
  2. Saisissez un nom.
  3. Sélectionnez la réponse du serveur publicitaire:
    • Règles publicitaires de DFP
    • VMAP DFP
    • XML intelligent
    • VASTE 3.0
  4. Saisissez la balise d'annonce.
  5. Cochez Désactiver les balises côté serveur pour désactiver les balises côté serveur. Lorsqu'il est désactivé, SSAI ne déclenche aucun balises côté serveur et inclura toutes les balises dans la sortie VMAP.
  6. Cliquez sur Save.

Pour modifier une configuration d'annonce, cliquez sur le nom de l'annonce, apportez les modifications, puis cliquez sur Enregistrer. Pour supprimer une configuration d'annonce, cochez la case en regard de l'annonce, puis cliquez sur Supprimer. Pour dupliquer une configuration d'annonce, cochez la case située en regard de l'annonce, puis cliquez sur Dupliquer.

Une fois les configurations d'annonces VOD créées, ouvrez le module Players et cliquez sur un lecteur pour afficher les propriétés. Lorsque la propriété de publicité Activer le côté serveur (SSAI) est activée, une configuration d'annonce peut être sélectionnée. Pour plus d'informations sur la configuration de la publicité des lectuers, voir Configuration de la publicité des lectuers à l'aide du module des lectuers.

Création d'une configuration Live Ad

Remarques:
  • Les configurations d'annonces Live côté serveur peuvent également être gérées à l'aide de l'API Live. Pour plus d'informations, consultez Brightcove Live API: Insertion d'annonces côté serveur (SSAI).

Pour ajouter une configuration Live Add, procédez comme suit.

  1. Dans la page Paramètres d'annonce côté serveur, cliquez sur Créer une configuration publicitaire > en direct.
  2. Saisissez un nom.
  3. Sélectionnez la réponse du serveur publicitaire:
    • VAST
    • DFP (Règles publicitaires)
    • XML intelligent
  4. Saisissez la balise d'annonce.
  5. Cliquez sur Save.

Pour modifier une configuration d'annonce, cliquez sur le nom de l'annonce, apportez les modifications, puis cliquez sur Enregistrer. Pour supprimer une configuration d'annonce, cochez la case en regard de l'annonce, puis cliquez sur Supprimer. Pour dupliquer une configuration d'annonce, cochez la case située en regard de l'annonce, puis cliquez sur Dupliquer.

Une fois les configurations d'annonces en direct créées, lorsqu'un événement est créé avec la propriété Activer l'insertion d'annonces côté serveur (SSAI) activée, une configuration d'annonce peut être sélectionnée dans la salle de contrôle. Pour plus d'informations sur la création d'événements en direct, voir Création et gestion d'événements en direct à l'aide du module Live.
