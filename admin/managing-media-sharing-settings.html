Le partage de médias permet aux éditeurs de charger des vidéos une première fois, puis de les partager sur d'autres comptes. Ces comptes peuvent être les vôtres, ceux de l'entreprise ou ceux d'autres éditeurs. Vous pouvez ainsi programmer et utiliser indépendamment les vidéos partagées sur d'autres comptes Video Cloud. Les paramètres de partage de médias sont utilisés pour gérer les relations de partage entre les comptes.

Note: Le partage multimédia n'est disponible que pour les clients Video Cloud Enterprise. Si vous souhaitez mettre à niveau votre compte Video Cloud, veuillez contacter Brightcove pour plus d'informations.

Le partage de médias peut également être implémenté par programme à l'aide de l'API CMS. Pour plus d'informations, consultez Partage de médias avec l'API CMS.

La terminologie suivante sera utilisée dans cette rubrique :

  • Master - Le compte qui a créé la vidéo originale. Le maître est propriétaire du contenu et est responsable de la configuration, de la gestion et de la fourniture du contenu aux Affiliés.
  • Affiliate - Le compte qui reçoit la vidéo. L'Affilié peut accepter le contenu qui lui est partagé à partir d'un Master.

Activation du partage de médias

Pour activer le partage de médias, contactez l'assistance client Brightcove et demandez-lui d'activer le partage de médias pour votre compte.

Ajout d'affiliés

Pour partager des vidéos sur un autre compte Video Cloud, un compte affilié doit être ajouté. Un affilié est un compte sur lequel vous souhaitez partager des vidéos. Pour ajouter un compte affilié, procédez comme suit :

  1. Connectez-vous à Video Cloud Studio en tant qu'administrateur.
  2. Dans l'en-tête de navigation, cliquez sur ADMIN et alors Partage médiatique. Notez que vous devez être un administrateur de compte pour accéder au menu ADMIN.
  3. Les Gérer les affiliés - Comptes avec lesquels je partage des vidéos La section affichera une liste de tous les éditeurs affiliés avec lesquels vous avez établi une relation de partage de médias.
  4. Pour ajouter un nouvel affilié, saisissez son ID de compte Video Cloud dans le champ identifiant de compte zone de texte, puis cliquez sur Ajouter un affilié.

Le compte sera ajouté à la table avec le statut de relation en attente. Le compte affilié recevra une notification de partage de médias et pourra accepter ou rejeter la relation. Si l'affilié accepte la relation de partage de médias, le statut de la relation sera Accepté.

Correspondance de champs personnalisés

Pour exiger que les comptes affiliés aient les mêmes champs personnalisés que le compte principal, cochez Appliquer la correspondance des champs personnalisés, puis cliquez sur Mettre à jour les règles. Lorsque l'option Appliquer la correspondance de champs personnalisés est cochée, les comptes d'affiliation doivent avoir les mêmes champs personnalisés que le compte principal, sinon les partages échouent.

Remarques

  • La correspondance des champs personnalisés vérifiera simplement que les noms de champs personnalisés internes dans les deux comptes correspondent. Le type du champ (liste ou texte) n'a pas d'importance.

  • Si le canal défini par le compte maître applique la correspondance des champs personnalisés, puis partage une vidéo comportant un champ personnalisé dont la valeur n'est pas autorisée par le compte d'affiliation, cette tentative de partage échouera.

    Si le canal défini par le compte maître n'applique pas la correspondance des champs personnalisés, puis partage une vidéo qui a un champ personnalisé avec une valeur qui n'est pas autorisée par le compte d'affiliation, alors la tentative de partage fonctionnera, mais le champ avec la valeur incorrecte ne sera pas inclus dans la copie d'affiliation de la vidéo.

  • Remarque: Si une vidéo partagée dans le compte maître est configurée avec le filtrage géo, les comptes d'affiliation doivent également être activés pour le filtrage géographique, sinon le partage échouera. Tout filtrage géo configuré sur la vidéo partagée sera également appliqué dans le (s) compte (s) affilié (s). Pour activer un compte pour le filtrage géographique, veuillez contacter l'assistance Brightcove.

Suppression des affiliés

Pour supprimer un compte affilié, localisez le compte à supprimer et cliquez sur l'icône de suppression (). Une boîte de dialogue s'affiche confirmant que vous souhaitez mettre fin à la relation de partage.

Cliquez sur Terminer le partage pour mettre fin à la relation de partage.

Approbation/rejet des relations de partage des médias

Si un autre Video Cloud a ajouté votre compte en tant qu'affilié (souhaite partager des vidéos sur votre compte Video Cloud), un message s'affiche lorsque le ADMIN> Partage multimédia page est accédée.

Si l'option Appliquer la correspondance de champs personnalisés est cochée, le compte d'affiliation doit avoir les mêmes champs personnalisés que le compte principal, sinon tous les partages échoueront.

Cliquez sur Rejeter pour rejeter la relation de partage de médias. Sinon, sélectionnez un mode d'acceptation:

  • Automatique - Acceptez toujours les vidéos partagées depuis ce compte principal. Les vidéos partagées apparaîtront dans la liste des vidéos du module Media.
  • Manuel: passez en revue les vidéos partagées de ce compte principal avant de les ajouter à la médiathèque. Le module Média affichera un Partages en attente élément dans la navigation de gauche.

Cliquez sur Approuver pour approuver la relation de partage de médias. Une fois la relation approuvée, elle apparaît dans la section Gérer les maîtres de la page.

Cliquez sur Afficher le contrat pour afficher les paramètres de partage de médias. Les Mode d'acceptation le curseur peut être utilisé pour changer le Mode d'acceptation. Pour supprimer la relation de partage, cliquez sur l'icône de suppression ().
