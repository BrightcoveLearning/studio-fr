La page d'administration des profils d'acquisition peut être utilisée pour créer de nouveaux profils d'acquisition hérités. Les profils d'ingest sont définis à l'aide de JSON. Lors de la création d'un nouveau profil hérité, il est utile de consulter le Profils d'ingestion standard pour le transcodage hérité document et recherchez un profil similaire à celui dont vous avez besoin. Ensuite, dans Studio, vous pouvez facilement dupliquer le profil, puis le modifier avec vos modifications.

Remarque: Pour plus d'informations sur la création de profils d'ingestion utilisant DRM, consultez le Vidéo MPEG-DASH document du développeur.

Remarque: Chaque profil peut créer un maximum de 25 formats associés.  Les images d'affiche et de vignettes comptent pour un maximum de 25. 

Pour créer un nouveau profil d'ingestion hérité :

  1. Cliquez sur le lien ADMIN dans l'en-tête Studio.
  2. Clique le Ingérer des profils relier.
  3. Recherchez un profil hérité à dupliquer. Cliquez sur le nom du profil pour ouvrir l'éditeur de profils.
  4. Cliquez sur Profil en double. L'éditeur de profil s'ouvrira avec le nouveau profil.
  5. Faites défiler la page vers le haut du profil. Modifiez l' name élément pour donner au profil un nom significatif. Ce nom s'affichera dans Video Cloud Studio.
  6. Modifiez l' description élément pour donner au profil une description significative.
  7. Modifiez les informations de format associé si nécessaire. La taille par défaut de l'affiche et des images miniatures se trouve à la fin du profil.
  8. Cliquez sur Save.

Ajouter un filigrane à un profil

Un filigrane peut être ajouté à chaque rendu en ajoutant l' watermarks élément à chacun rendition:

       filigranes": [{ "y": -20, "x": -10, "width": "20%", "url": "http://www.mysite.com/companylogo.png" }]

L' watermarks élément possède les paramètres suivants.

Paramètre Type Description
url chaîne URL de l'image du filigrane
width chaîne largeur en pixels ou en pourcentage de la largeur du cadre, par exemple 20 ou 10%
height chaîne hauteur en pixels ou en pourcentage de la largeur du cadre, par exemple 20 ou 10 %
x chaîne distance entre le bord gauche et le centre de l'image en pixels ou en pourcentage de la largeur de l'image ; par exemple 20 ou 10 % (utilisez des valeurs négatives telles que -10 pour positionner le filigrane par rapport au bord droit)
y chaîne distance entre le bord supérieur et le centre si l'image est exprimée en pixels ou en pourcentage de la hauteur de l'image ; par exemple 20 ou 10 % (utilisez des valeurs négatives telles que -10 pour positionner le filigrane par rapport au bord inférieur)
