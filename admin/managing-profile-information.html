Les utilisateurs de Studio peuvent utiliser la page Informations sur le profil pour afficher et modifier leurs propres informations de compte personnel.

Pour accéder à la page Profile Information :

  1. Connectez-vous à Studio.
  2. Dans la partie droite de l'en-tête de navigation, cliquez sur le sélecteur de compte, puis sur le Mon profil relier.

Modification des informations sur le profil

Tous les champs de la section Profile Information peuvent être modifiés. Cliquez sur Save Changes pour enregistrer les modifications apportées.

Remarques relatives à la modification :

  • Vous ne pouvez pas modifier votre adresse e-mail. Si vous devez modifier votre adresse e-mail, un compte avec la bonne adresse doit être créé. L'administrateur du compte principal peut le faire.
  • Vous pouvez modifier votre mot de passe, mais vous devrez renseigner votre mot de passe actuel.
  • Les mots de passe ne doivent pas dépasser 8 lettres et contenir au moins un chiffre et une lettre
  • Le fuseau horaire que vous définissez ici est utilisé lorsque vous programmez la date et l'heure d'activation d'une vidéo. Vous pouvez planifier la date de début et de fin d'activation d'une vidéo en accédant à Studio Médias module, en sélectionnant les détails d'une vidéo, puis en éditant le Disponibilité section.
  • Vous pouvez utiliser la liste déroulante Langue pour sélectionner la langue Studio :
    • Anglais
    • Espagnol (Espagne)
    • Japonais
    • Français (France)
    • Chinois (traditionnel)
    • Coréen
Note: Si vous êtes un administrateur, vous pouvez utiliser le Gestion des utilisateurs dans Studio pour modifier le mot de passe et le rôle d'un utilisateur.
