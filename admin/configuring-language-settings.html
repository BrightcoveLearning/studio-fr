Les paramètres de langue sont l'endroit où vous définissez une langue principale, puis ajoutez et gérez des langages de métadonnées supplémentaires. La langue principale est la langue que votre compte utilise actuellement pour les métadonnées vidéo. Avant d'ajouter des langues supplémentaires à votre compte, vos métadonnées existantes seront associées à la langue que vous définissez comme langue principale. La langue principale est ce qui s'affiche initialement lorsque vous modifiez les propriétés vidéo dans le module Media.

Définition d'une langue principale

Pour définir la langue principale :

  1. Connectez-vous à Video Cloud Studio.
  2. Dans l'en-tête de navigation, cliquez sur ADMIN et alors Paramètres de langue. Notez que vous devez être un administrateur de compte pour accéder au menu ADMIN.
  3. Entrez votre langue principale. Au fur et à mesure que vous tapez, une liste des langues correspondantes s'affiche.
  4. Cliquez sur une langue pour la sélectionner.
  5. Cliquez sur Enregistrer la langue.
Pour changer la langue principale, cliquez sur la langue, sélectionnez une nouvelle langue puis cliquez sur sauver.

Ajout de langues supplémentaires

Une fois la langue principale enregistrée, d'autres langages de métadonnées peuvent être ajoutés. En ajoutant des langues supplémentaires, vous pourrez entrer des métadonnées multilingues supplémentaires dans le module Media. Pour ajouter des langues supplémentaires, procédez comme suit.

  1. Cliquez sur Ajouter une langue.
  2. Entrez votre langue. Au fur et à mesure que vous tapez, une liste des langues correspondantes s'affiche.
  3. Cliquez sur une langue pour la sélectionner.
  4. Cliquez sur Enregistrer la langue. La langue sera ajoutée au tableau Langues supplémentaires.
  5. Cliquez sur Ajouter une langue pour ajouter des langues supplémentaires si nécessaire.

Pour modifier le nom de la langue (tel qu'il apparaîtra dans le module Médias), cliquez dessus, effectuez les modifications nécessaires puis cliquez sur sauver. Pour supprimer une langue, cliquez sur l'icône de suppression ().

Note: Si vous supprimez une langue utilisée, toutes les métadonnées associées à la langue seront masquées. Pour restaurer les métadonnées associées, ajoutez à nouveau la langue.
