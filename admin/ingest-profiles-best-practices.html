Combien de profils dois-je avoir ?

Vous pouvez définir jusqu'à 200 profils personnalisés. Cependant, vous devez généralement en créer seulement autant que vous le souhaitez, afin d'éviter toute confusion lors de la sélection des profils pour les emplois d'ingestion.

Comment gérer mes profils ?

Vous pouvez utiliser le Page de gestion des profils d'acquisition dans Studio pour gérer vos profils d'acquisition. Diverses options sont disponibles dans cette interface, y compris :

  • Définition de votre profil par défaut
  • Création de nouveaux profils
  • Clonage à partir de profils existants
  • Modification de profils personnalisés
  • Suppression des profils inutiles

Si vous avez besoin d'un accès par programme à vos profils d'acquisition, vous pouvez créer, lire, mettre à jour et supprimer des profils personnalisés à l'aide du API REST Ingest Profiles. Notez que cela vous oblige à configurer Identifiants OAuth.

Comment mettre à jour mes profils ?

Voir le section ci-dessus.

Combien de formats associés un profil peut-il avoir ?

UNE Livraison dynamique Le profil peut avoir jusqu'à 37 rendus. Notez que pour les profils de diffusion dynamique personnalisés, vous ne pouvez sélectionner que rendus prédéfinis qui sont actuellement disponibles.

Si vous créez un profil d'encodage contextuel personnalisé, min_renditions a une valeur minimale de 1, et max_renditions a une valeur maximale de 15.

Notez que les formats associés inutiles augmenteront vos coûts de stockage géré. Vous ne devez donc pas créer de formats associés qui sont peu susceptibles d'être lus compte tenu du type de vidéos que vous publiez et des audiences et des appareils que vous ciblez.

Les profils standard de votre compte sont conçus pour offrir de bonnes expériences visuelles sur les bandes passantes typiques des visionneuses et les appareils les plus couramment utilisés.

Nommage des profils personnalisés

Les noms de profil personnalisés ne doivent pas inclure d'espaces ou de caractères spéciaux. Utilisez des lettres et des chiffres et remplacez les espaces par des tirets (-) ou des traits de soulignement (_).

Pour éviter les erreurs des utilisateurs, essayez de nommer les profils d'une manière qui communiquera clairement aux personnes qui téléchargent des vidéos à quoi le profil doit être utilisé.

Quel profil dois-je définir par défaut ?

Tout profil standard ou personnalisé peut être défini par défaut dans Studio ou via l'API Ingest Profile. La valeur par défaut ne sera utilisée que si l'utilisateur ne spécifie pas de profil particulier (soit dans le module Upload, soit à l'aide de l'API Dynamic Ingest).

Nous vous recommandons de définir le profil par défaut sur un profil qui ne serait pas un très mauvais choix si l'utilisateur oublie de spécifier un profil particulier.
