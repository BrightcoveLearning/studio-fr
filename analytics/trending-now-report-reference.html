Note: Cette fonctionnalité est actuellement en version bêta. Si vous souhaitez l'activer pour votre compte, contactez votre responsable de compte.

Pour afficher le rapport Tendances maintenant, ouvrez le module Analytics et sélectionnez Tendances maintenant dans la navigation de gauche. Le rapport affiche un graphique indiquant le nombre de visionneuses simultanées au cours de la dernière heure. Le graphique sera mis à jour toutes les 5 secondes.

Note: Un utilisateur unique est déterminé par l'agent utilisateur du navigateur combiné avec l'adresse IP. Plusieurs onglets du même lecteur exécutés dans le même navigateur ne seront qu'un seul utilisateur.

Le rapport Trending Now utilise le fenêtrage spéculatif pour calculer la valeur partiellement complétée. Ces agrégations émettent une nouvelle valeur minutieusement toutes les 1 minute puis une valeur complète à la fin de la minute. En fonction de la charge du système, le processus d'écrasement peut ne pas être terminé, ce qui entraîne des pics d'une minute dans le rapport CCU.

Si votre compte est activé pour Brightcove Live, vous avez la possibilité d'afficher le nombre de visionneuses simultanées pour toutes les vidéos, ressources VOD ou actifs en direct.

Les sections sous le graphique afficheront les meilleures vidéos, les pays les plus populaires et les villes les plus populaires pour la dernière minute et seront mises à jour toutes les 5 secondes.
