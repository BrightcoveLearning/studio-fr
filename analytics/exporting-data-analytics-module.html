Exportation des données de rapports

Pour exporter les données d'un rapport, procédez comme suit.

  1. Accédez au module de statistiques.
  2. Utilisez le sélecteur de date pour définir une plage temporelle.
  3. Sélectionnez un rapport.
  4. Cliquez sur le lien approprié pour exporter les données dans un fichier CSV ou XLSX.
  5. Enregistrez le fichier de données.
