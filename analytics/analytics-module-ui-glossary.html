Note: Pour obtenir la liste des éléments de données renvoyés par l'API Analytics et lors de l'exportation de données analytiques, reportez-vous au Glossaire de l'API Analytics.

A

Médias actifs
Nombre de vidéos actives dans le compte. Notez que ces données ne sont pas en temps réel: elles sont synchronisées à partir de votre vidéothèque une fois par jour. Notez également que ces données peuvent ne pas être exactes si vous regardez des données analytiques « en temps réel » (datant de moins de 3 jours).
API Analytics (Statistiques)
Interface de programmation d'applications basée sur REST qui permet aux développeurs de récupérer des mesures d'analyse pour les vidéos Video Cloud directement au format JSON. Les API d'analyse fournit les données pour l'interface utilisateur d'analyse dans Video Cloud Studio.

B

Total Bandwidth
Bande passante totale utilisée pour fournir des vidéos, des images et des charges de lecteur. Notez que certaines parties des données de bande passante sont retardées, car elles proviennent de sources externes telles que les CDN. Normalement, l'information est complète dans les 48 heures, mais dans certains cas, la journée peut être jusqu'à 5 jours.
Type de navigateur
Le type de brower. Les valeurs typiques sont les suivantes :
  • chrome
  • safari
  • firefox
  • c'est-à-dire
Octets fournis
Données totales, en octets, fournies aux spectateurs, y compris les données vidéo/audio (y compris les données tamponnées qu'elles aient été lues ou non), les images d'affiche et de vignettes, et toutes les données associées au lecteur vidéo.

C

Ville
La ville dans laquelle une vidéo a été vue.
Pays
Abréviation anglaise du pays où la vidéo a été vue. Suit la norme ISO 3166.
CSV
Valeurs séparées par des virgules. Vous pouvez exporter les données dans ce format.

D

Daily Unique Users
Le nombre estimé d'utilisateurs uniques qui ont vu une vidéo pour un jour donné. Ces données ne sont disponibles que des données rapprochées (après 3 jours), et uniquement au niveau du compte. Pour plus d'informations, voir API Analytics: Utilisateurs uniques.
Jour
Le jour du mois pendant lequel la vidéo a été vue.
Dispositif
Combinaison du type d'appareil et du système d'exploitation de l'appareil utilisé pour regarder la vidéo.
Device Operating Systems
Le système d'exploitation utilisé sur l'appareil qui affiche la vidéo. Les valeurs acceptées sont :
  • Android
  • Bada
  • iOS
  • Linux
  • OS X
  • RIM (Mûre)
  • Symbian
  • Web OS
  • Windows
  • Other: pour les ressources qui ne rentrent dans aucune catégorie ci-dessus
Fabricant d'appareils
Le fabricant de l'appareil. Voici quelques valeurs typiques :
  • lenovo
  • apple
  • sony
  • microsoft
Type de dispositif
Le type d'appareil utilisé pour regarder une vidéo. Les valeurs acceptées sont :
  • Desktop: comprend les ordinateurs de bureau et les ordinateurs portables
  • Tablet: comprend les iPad, les tablettes sous Android, etc.
  • Mobile: téléphones mobiles
  • Connected TV/Game Console/Companion Device: comprend les TV connectées, les box de salon et les consoles de jeu
  • Other: pour les ressources qui ne rentrent dans aucune catégorie ci-dessus
 
Dimension
Les échelles (Dimension) sont des catégories de données statistiques que vous pouvez filtrer et obtenir sous forme de rapports. Les échelles actuellement disponibles sont :
  • lectuer
  • Vidéo
  • Pays
  • Jour
  • Afficher le domaine
  • Appareil
  • Type d'appareil
  • Système d'exploitation de l'appareil
  • Source de trafic
 
Domaine d'affichage
Le domaine de référence communiqué par le client (navigateur ou autre).
Afficher URL
L'URL où se trouvait le spectateur lorsqu'il a regardé la vidéo.

E

Note d'engagement
Le pourcentage moyen de la vidéo vue par visionnage (calculé en additionnant tous les pourcentages regardés et en divisant le résultat par le nombre de vues de la vidéo). Pour plus d'informations sur le calcul de cette statistique, consultez l'article Calcul de l'engagement vidéo.

Taux de lecture
Le taux de lecture est le nombre de fois qu'une vidéo a été lue (video_view) divisé par le nombre d'opportunités de lecture (video_impression).
Demande de lecture
Une demande de lecture est comptée à chaque fois que le bouton de lecture est cliqué.
lectuer
L'ID du lecteur Video Cloud
Chargement du lecteur
Le nombre de fois où un lecteur a été chargé.
Nom du lecteur

R

Région
Le nom de l'état ou de la province où la vidéo a été regardée. Suit la norme ISO 3166.

T

Traffic Sources
Une indication sur la façon dont le spectateur est arrivé sur la page du lecteur. Les valeurs acceptées sont :
  • Referral: l'utilisateur a cliqué sur un lien vers la vidéo.
  • Direct - L'utilisateur a directement tapé le lien, utilisé un signet ou utilisé un lien provenant d'un e-mail.
  • Recherche organique - L'utilisateur a effectué une recherche et a cliqué sur un résultat pour accéder à la vidéo
  • Paid Search (Recherche sponsorisée): l'utilisateur a lancé une recherche et a cliqué sur un lien sponsorisé pour afficher la vidéo
  • Secure Search - L'utilisateur s'est connecté avec un compte Google, a effectué une recherche et a cliqué sur un résultat (payant ou organique ; les informations de recherche ne sont pas incluses dans ce cas)

U

Unique Users
Voir Daily Unique Users

V

Vidéo
L'ID Video Cloud de la vidéo.
Bande passante vidéo
Bande passante totale utilisée pour diffuser des vidéos uniquement.
Durée de la vidéo
La longueur de la vidéo en secondes.
Engagement vidéo
Le pourcentage moyen de la vidéo vue par visionnage (calculé en additionnant tous les pourcentages regardés et en divisant le résultat par le nombre de vues de la vidéo). Pour plus d'informations sur le calcul de cette statistique, consultez l'article Calcul de l'engagement vidéo.
Nom de la vidéo
Le nom de la vidéo.
Visionnages de la vidéo
Le nombre de fois où la lecture de la vidéo a été lancée. Cette statistique est recueillie au lancement du flux et ne comprend pas les retours en arrière ou les relectures. Cette statistique ne mesure pas le nombre de spectateurs individuels.
Nombre de minutes vues
Le nombre total de minutes vues d'une vidéo.
Vues à 1%
Nombre de vues enregistrées au point de 1% de la vidéo.
Vues à 25%
Nombre de vues enregistrées au point de 25% de la vidéo.
Vues à 50%
Le nombre de vues enregistrées aux 50 % de la longueur de la vidéo.
Vues à 75%
Le nombre de vues enregistrées aux 75 % de la longueur de la vidéo.
Vues à 100 %
Le nombre de vues enregistrées aux 100 % de la longueur de la vidéo.
Video Impression
Le nombre de fois où une vidéo a été chargée dans un lecteur et a été prête pour interaction.
