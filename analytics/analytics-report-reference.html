Le module Video Cloud Analytics fournit les rapports suivants :

  • Performance
  • Engagement
  • À la mode maintenant
  • Personnalisé

Vous trouverez ci-après un résumé de chacun des rapports.

Rapport Performance

Le rapport sur les performances indique les performances des vidéos de votre compte, notamment les vues vidéo, les impressions de vidéos, le taux de lecture, les minutes consultées et le score d'engagement. Le rapport de performance permet de segmenter les données en dimensions :

  • Video: affiche les mesures de performances des vidéos.
  • Player: affiche les mesures de performances des lecteurs.
  • Source of Traffic: affiche les mesures de performances par source de trafic.
  • Domaine d'affichage : affiche les mesures de performances en fonction du domaine sur lequel se trouvait le visualiseur lors de la visualisation d'une vidéo. Si le lecteur se trouve dans une iframe, le domaine d'affichage sera enregistré en tant qu'URL de la page dans l'iframe. (Cela correspond à la destination_domain dimension de l'API Analytics.)
  • Country: affiche les performances par pays de visionnage.
  • Appareil: affiche les mesures de performance pour chaque type de périphérique - Combinaison de système d'exploitation de périphérique utilisée pour visionner une vidéo
  • Device Type: affiche les mesures de performances par type de dispositif.
  • Système d'exploitation des appareils: affiche les mesures de performance par système d'exploitation de l'appareil
  • Fabricant de périphériques - Affiche les mesures de performance par fabricant de l'appareil
  • Type de navigateur: affiche les mesures de performance par type de navigateur
  • Day: affiche les performances d'un jour donné.

Pour plus d'informations sur le rapport Performance, consultez la référence pour les rapports de performances.

Rapport Engagement

Le rapport d'engagement indique l'engagement pour les vidéos de votre compte. Il comprend une note d'engagement et le nombre de vues à 1, 25, 50, 75 et 100 % de la vidéo. Le rapport d'engagement permet de segmenter les données en dimensions :

  • Vidéo - Affiche les mesures d'engagement sur les vidéos
  • Player - Affiche les mesures d'engagement sur les lectuers
  • Day - Affiche les mesures d'engagement par jour

Pour plus d'informations sur le rapport Engagement, consultez la référence pour les rapports sur l'engagement.

Rapport sur les tendances actuelles

Le rapport Tendances actuelles affichera un graphique indiquant le nombre de spectateurs simultanés au cours de la dernière heure. Le rapport Trending Now peut être utilisé pour afficher les données de la VOD, des vidéos en direct ou des deux types de vidéos.

Pour obtenir des informations détaillées sur le rapport sur les tendances actuelles, consultez le Référence du rapport sur les tendances actuelles.

Rapports personnalisés

Le module Analytics peut être utilisé pour générer des rapports analytiques personnalisés et ad hoc qui peuvent être téléchargés au format de valeurs séparées par des virgules (.csv) ou Microsoft Excel (.xlsx).

Note: Les rapports personnalisés sont disponibles pour les éditeurs Enterprise et Pro.

Pour plus d'informations sur la création de rapports personnalisés, consultez le Création de rapports d'analyse personnalisés dans le module d'analyse.

Exportation des données de rapports

Les données de rapports peuvent être téléchargées en tant que valeurs séparées par des virgules et des formats Excel.
