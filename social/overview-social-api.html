Introduction

L'API Social permet d'obtenir le statut et l'historique du partage social pour vos vidéos. Il est principalement destiné à aider avec les intégrations CMS, pour permettre à vos utilisateurs de visualiser ces informations dans votre propre système CMS.

Pour plus de détails sur les requêtes, reportez-vous à la référence de l'API.

URL de base

L'URL de base de l'API est :

  https://edge.social.api.brightcove.com/v1

Authentification

L'authentification des demandes nécessite un En-tête d'autorisation :

        Authorization: Bearer {access_token}

Les access_token est un jeton d'accès OAuth2 temporaire qui doit être obtenu auprès du Brightcove OAuth service. Pour plus de détails sur la façon d'obtenir les informations d'identification client et de les utiliser pour récupérer des jetons d'accès, consultez le Présentation de Brightcove OAuth sujet.

Opérations

Lorsque vous demandez des informations d'identification client, vous devez spécifier le type d'accès au compte ou les opérations que vous souhaitez. Ce qui suit est une liste des opérations actuellement prises en charge pour le API sociale :

  • video-cloud/social/status/read

Notez que lorsque l'API sociale est en version bêta, les autorisations ne sont pas disponibles dans la section Administrateur Studio pour la gestion des informations d'identification de l'API. Au lieu de cela, vous devrez utiliser l'API OAuth pour créer vos informations d'identification. Cependant, Brightcove Learning Services, a créé un outil en ligne simple pour ce faire.

Résumé des demandes

Dans la version actuelle de l'API, seules les GET requêtes sont prises en charge. Il y a deux groupes de demandes, résumés dans les sections suivantes.

Demandes d'état

Les demandes d'état renvoient l'état actuel du partage social pour les vidéos. Vous pouvez obtenir le statut de toutes les vidéos de votre compte ou d'une vidéo spécifique.

Demandes d'historique

Les demandes d'historique renvoient l'historique de partage social pour les vidéos. Vous pouvez obtenir l'historique de toutes les vidéos de votre compte, ou une vidéo spécifique.
