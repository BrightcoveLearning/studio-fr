Brightcove Social fournit plusieurs rapports qui peuvent être utilisés pour surveiller les performances de vos vidéos publiées sur vos comptes de médias sociaux :

  • Tableau de bord: offre une vue d'ensemble de haut niveau des performances vidéo sur les comptes de médias sociaux. Pour plus d'informations, voir Consulter Social Analytics.
  • Social Views - Fournit une vue d'ensemble de la vue sociale des vidéos partagées sur des comptes de médias sociaux. Couvert dans cette rubrique.
  • Engagement social - Fournit un aperçu de l'engagement social (J'aime, partages, Retweets, etc.) pour les vidéos partagées sur des comptes de médias sociaux. Pour plus d'informations, voir Consulter Social Engagement Analytics.

Commencez par sélectionner une période pour le rapport. Pour afficher le rapport Social Views Analytics, cliquez sur l'onglet Vues sociales en haut de la page.

Le rapport Social View Analytics est organisé dans les sections suivantes :

  • Points de vue sociaux
  • Détails des affichages sociaux

Points de vue sociaux

Le graphique Vues sociales affiche le nombre de vues vidéo pour les vidéos publiées sur vos comptes de médias sociaux pour la période sélectionnée. La date de mise à jour est la dernière date pour laquelle Video Cloud dispose de données d'analyse sociale. Généralement, il y a un délai de 24 à 48 heures avant que les plates-formes sociales mettent à jour les vues et les données d'engagement dans leurs API.

Note: Les API Twitter Insights ne signalent pas les vues des médias, par conséquent, la visualisation des données n'est pas disponible dans le cadre de l'analyse sociale.

La légende à droite du graphique permet de sélectionner le niveau de détail à afficher :

  • Mes sites et applications - Afficher le nombre de vues vidéo depuis les lecteurs et applications Video Cloud
  • Facebook - Afficher le nombre de visionneages vidéo sur Facebook
  • YouTube - Afficher le nombre de visionnements vidéo sur YouTube
  • Total: affiche le nombre total de vues vidéo sur toutes les plateformes

Le survol d'une ligne dans le graphique affichera les données de cette date.

Détails des vues sociales

La section Détails des vues sociales peut être utilisée pour afficher les détails de la vue vidéo en :

  • Vues (par défaut)
  • Temps moyen de surveillance
  • Pourcentage moyen surveillé

Une fois la sélection effectuée, le rapport affiche les données détaillées pour chaque vidéo.

Note: Les API Twitter Insights ne signalent pas les vues des médias, par conséquent, la visualisation des données n'est pas disponible dans le cadre de l'analyse sociale.

Les vues YouTube affichées dans les rapports d'analyse sont collectées à partir du Présentation de YouTube Analytics rapport. YouTube ne garantit pas que ces données sont des données en temps réel. Par conséquent, vous pouvez voir des retards de plusieurs jours avant que les vues et l'engagement apparaissent dans ce rapport. Une fois que les données sont dans YouTube Analytics, elles seront récupérées par les collecteurs de données nocturnes pour Brightcove Social Analytics.

Si vous avez besoin de données de visualisation en temps réel plus récentes, YouTube fournit également des rapports en temps réel avec des estimations des données de visionnage au cours des dernières 48 heures pour vos 25 vidéos les plus récentes publiées. Vous pouvez trouver le rapport en temps réel de votre chaîne YouTube ici. Brightcove Social Analytics ne collecte pas ces données en temps réel car elle fournit des vues estimées (par opposition aux données réelles) et ne s'applique qu'aux vidéos récentes.

Au bas du rapport, vous trouverez des contrôles qui vous permettent de sélectionner le nombre de lignes à afficher et de naviguer vers d'autres pages. Cliquez sur le lien correspondant à l'un des titres vidéo pour afficher l'analyse des vues sociales pour la vidéo sélectionnée.

Questions courantes

1. Lorsque le module Social est utilisé pour publier une vidéo sur YouTube, puis que cette vidéo est intégrée dans un lecteur YouTube sur un autre site, les données de lecture/engagement seront-elles toujours collectées et transmises au module Social analytique ?

Social Analytics recueille toutes les données de vue signalées par YouTube, telles qu'elles apparaissent dans vos pages Creator Studio et/ou Analytics sur YouTube. Si les vues du lecteur intégré YouTube sont prises en compte dans vos analyses, le module Social le capture et le signale. L'utilisation du code d'intégration de la fonctionnalité Partager > Intégrer de YouTube exactement comme indiqué permet de compter les vues de ce lecteur intégré dans vos analyses dans YouTube. Cependant, la modification du code intégré peut entraîner YouTube à ne pas collecter les données de vues, et en particulier la configuration du lecteur intégré pour qu'il lise automatiquement la vidéo semble faire en sorte que YouTube ne comptera pas les vues de ce lecteur.

Exportation des données

Les données de rapports peuvent être téléchargées en tant que valeurs séparées par des virgules et des formats Excel. Notez que la fonction d'exportation enregistre les données directement à partir d'un API d'analyse appel. Les données exportées ne sont pas modifiées comme les données de l'interface utilisateur. Elles contiennent actuellement toutes les données inhérentes au rapport et pas seulement le sous-ensemble de champs de l'interface utilisateur. Obtenir toutes les données au format de base depuis l'API vous permet de manipuler les données via d'autres méthodes que celles de l'interface utilisateur.
