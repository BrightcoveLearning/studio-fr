Pour accéder à Brightcove Social, connectez-vous à Video Cloud Studio et cliquez sur Social dans l'en-tête de navigation. Une liste de vos vidéos publiées sera affichée par défaut.

En utilisant Brightcove Social, vous pouvez :

  • Voir vos vidéos publiées
  • Publier des vidéos sur les plateformes de médias sociaux
  • Synchronisation automatique des vidéos sur les comptes de médias sociaux
  • Créer des destinations de publication sur les médias sociaux
  • Revoir Social Analytics

Remarque: nous avons rencontré certains cas où le module Social ne se charge pas. Le plus souvent, cela semble être causé par le réseau de l'utilisateur bloquant la bibliothèque Facebook : https://connect.facebook.net/en_US/sdk.js.

Ceci est particulièrement courant dans Firefox, en raison de la protection améliorée contre le pistage, qui bloque le SDK Facebook par défaut. Pour obtenir des informations et des instructions sur le réglage de ces paramètres, reportez-vous à Protection contre le pistage améliorée dans Firefox pour ordinateur de bureau.

Visualisation de vos vidéos publiées

Pour afficher vos vidéos publiées, cliquez sur le lien Published dans la navigation de gauche. Chaque vidéo publiée affichera l'état de publication, les informations vidéo et un lien vers la plateforme sociale sur laquelle la vidéo a été publiée. Pour plus d'informations sur la visualisation des vidéos publiées, reportez-vous à la section Affichage des vidéos publiées sur les réseaux sociaux.

Publication de vidéos sur les plates-formes de médias sociaux

Pour publier une vidéo sur une plate-forme de médias sociaux, cliquez sur le lien Videos dans le volet de navigation de gauche. Pour obtenir des informations sur la publication de vidéos sur un compte de médias sociaux, consultez la rubrique Publication de vidéos sur les plates-formes de médias sociaux.

Synchronisation automatique des vidéos sur les comptes de médias sociaux

La fonctionnalité Auto Sync sert à publier des vidéos sur les médias sociaux de façon automatique en leur affectant des balises. Pour configurer Auto Sync, cliquez sur Auto Sync dans le volet de navigation de gauche. Pour plus d'informations sur la configuration de la fonctionnalité Synchronisation automatique, voir Utilisation de la synchronisation automatique pour synchroniser des vidéos sur les comptes de médias sociaux.

Création de destinations de publication sur les médias sociaux

Les destinations de publication sont les comptes de médias sociaux sur lesquels vous publiez les vidéos. Il peut s'agir d'une chaîne YouTube, d'une page Facebook ou d'un compte Twitter. Pour créer une destination de publication, cliquez sur Destinations dans le volet de navigation de gauche. Pour obtenir des informations sur la création de destinations de publication, consultez la rubrique Création de destinations de publication sur les médias sociaux.

Passage en revue des statistiques vidéo relatives aux médias sociaux

La fonctionnalité Social Analytics fournit des données sur la performance des vidéos publiées sur les chaînes de médias sociaux. Pour afficher les statistiques relatives aux médias sociaux, cliquez sur Social Analytics dans le volet de navigation de gauche. Pour obtenir des informations sur les statistiques vidéo relatives aux médias sociaux, consultez la section Passage en revue des statistiques vidéo relatives aux médias sociaux.
