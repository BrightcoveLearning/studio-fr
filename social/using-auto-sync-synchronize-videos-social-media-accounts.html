Brightcove Social fournit une fonction de synchronisation automatique qui peut être utilisée pour synchroniser automatiquement les vidéos sur les réseaux sociaux en fonction des balises associées à une vidéo.

Avant de configurer la synchronisation automatique, certaines destinations de publication doivent être créées. Pour obtenir des informations sur la création de destinations de publication, consultez la rubrique Création de destinations de publication sur les médias sociaux.

Notez que l'utilisation de la synchronisation automatique et la publication manuelle de vidéos à l'aide du module Social sont des opérations indépendantes. Auto Sync ne sait pas quelles vidéos ont été publiées manuellement à l'aide du module Social. Si une vidéo est balisée de manière à ce que Auto Sync la publie dans l'une de vos destinations sociales, elle le fera indépendamment du fait que cette vidéo ait déjà été publiée manuellement à l'aide du module Social.

Configuration de la synchronisation automatique

Pour configurer la synchronisation automatique, procédez comme suit.

Une fois les balises entrées, le nombre de vidéos avec ces balises sera affiché.

Note: Les vidéos existantes avec les balises spécifiées ne seront pas automatiquement synchronisées. Toutes les nouvelles vidéos avec ces balises seront automatiquement synchronisées. Pour synchroniser vos vidéos existantes, modifiez les métadonnées vidéo pour déclencher une synchronisation automatique.
  1. Ouvrez Brightcove Social.
  2. Cliquez sur Synchronisation automatique dans la navigation de gauche. Une liste de tous les noms de synchronisation automatique qui ont été créés s'affiche.
    Note: L'image miniature sera renseignée une fois les vidéos synchronisées.
  3. Cliquez sur Configurer la synchronisation automatique.
  4. Entrez un nom pour la synchronisation.
  5. Sélectionnez les vidéos à distribuer.
    • Vidéos avec tous ces tags - Les vidéos contenant TOUTES les balises répertoriées doivent être synchronisées
    • Vidéos comportant l'une de ces balises - Les vidéos comportant N'importe laquelle des balises répertoriées doivent être synchronisées
    • Toutes les vidéos de ce compte - Toutes les vidéos du compte doivent être synchronisées
  6. À moins que toutes les vidéos de ce compte aient été sélectionnées, entrez une liste de balises séparées par des virgules.
  7. Sélectionnez toutes les vidéos qui doivent être exclues.
    • Inactif - Les vidéos inactives ne seront pas synchronisées (par défaut)
    • Programmation restreinte - Les vidéos programmées ne seront pas synchronisées
    • Restrictions géographiques: les vidéos avec des restrictions géographiques ne seront pas synchronisées
  8. Cliquez sur Next (Suivant).
  9. Choisissez une destination de publication vers laquelle synchroniser les vidéos. Cliquez sur Gérer les destinations pour créer de nouvelles destinations ou modifier des destinations existantes. Cliquez sur une destination pour afficher les valeurs par défaut de destination.
    Note: Les valeurs par défaut de destination peuvent être modifiées en modifiant les propriétés de destination.
  10. Cliquez sur Next (Suivant).
  11. Configurez les règles de synchronisation automatique. Les options disponibles changeront en fonction de la plateforme sociale sélectionnée.

Configuration des règles de synchronisation automatique

Lors de la configuration de la synchronisation automatique, des règles peuvent être configurées pour gérer ce qui arrive aux vidéos sur le site de médias sociaux. Cela inclut les mesures à prendre lorsque la vidéo est supprimée de la liste (la vidéo est supprimée de Video Cloud ou les balises changent) ou comment les mises à jour des métadonnées vidéo sont gérées. Les règles sont basées sur la plateforme de médias sociaux qui a été sélectionnée :

  • YouTube
  • Facebook
  • Twitter
Notes :
  • Une fois les vidéos publiées sur une plateforme sociale à l'aide de la synchronisation automatique, Brightcove Social ne suit pas les actions qui se produisent sur cette vidéo du côté des médias sociaux. Par exemple, Brightcove Social n'est pas averti lorsqu'une vidéo a été supprimée d'une plateforme sociale et que le fait que la vidéo a été supprimée ne sera pas reflété dans Brightcove Social. Cela peut également se produire si une plateforme de médias sociaux détecte du contenu protégé par le droit d'auteur puis le supprime. Toute modification ultérieure de la vidéo du côté Video Cloud entraînant un autre événement de synchronisation entraînera des erreurs de synchronisation dans Brightcove Social. Dans ce cas, il s'agit d'une « erreur rétryable », et la nouvelle tentative entraînera le téléchargement de la vidéo avec les modifications appliquées.
  • Il y a une limite initiale de 20 autosynchronisations possibles disponibles. Si vous en avez besoin, veuillez contacter le support Brightcove.

Configuration des règles YouTube

Lors de l'utilisation de la synchronisation automatique avec YouTube :

  • YouTube a une limite de 100 caractères pour les titres de sorte que le module Social tronquera les titres vidéo à 100 caractères.
  • Si l' option Mettre à jour les métadonnées vidéo sur YouTube est sélectionnée, si la vidéo est téléchargée sans sous-titres et que des légendes sont ajoutées, les légendes seront transmises à YouTube.
  • Si l' option Mettre à jour les métadonnées vidéo sur YouTube est sélectionnée, si la vidéo est téléchargée avec des légendes, toute modification des informations sur les sous-titres dans Video Cloud entraînera la transmission des sous-titres mis à jour à YouTube.
  1. Sélectionnez ce qui se passe lorsqu'une vidéo n'est pas balisée ou supprimée de Video Cloud :
    • Supprimer une vidéo distribuée sur YouTube - La vidéo sera supprimée de YouTube
    • Créer une vidéo distribuée - Rendre la vidéo YouTube privée ou non répertoriée
    • Aucune action - Rien n'arrive à la vidéo YouTube
  2. Sélectionnez ce qui se passe lorsque les métadonnées vidéo sont modifiées :
    • Mettre à jour les métadonnées vidéo sur YouTube - Les modifications apportées aux métadonnées vidéo du côté Video Cloud seront synchronisées sur YouTube
    • Pas d'action - Aucune mesure n'est prise
      Noter: Lorsque le fichier source vidéo est mis à jour dans Studio, la vidéo mise à jour n'est pas synchronisée avec YouTube. Il s'agit d'une limitation de YouTube.
  3. Cliquez sur Next (Suivant).
  4. Confirmez les paramètres de synchronisation automatique, puis cliquez sur Terminer.

Configuration des règles Facebook

  1. Sélectionnez ce qui se passe lorsqu'une vidéo n'est pas balisée ou supprimée de Video Cloud :
    • Supprimer la vidéo distribuée sur Facebook - La vidéo sera supprimée de Facebook
    • Expire la vidéo distribuée sur Facebook - La vidéo expirera sur Facebook
    • Aucune action - Rien n'arrive à la vidéo Facebook
  2. Sélectionnez ce qui se passe lorsque les métadonnées vidéo sont modifiées :
    • Mettre à jour les métadonnées vidéo sur Facebook - Les modifications apportées aux métadonnées vidéo du côté Video Cloud seront synchronisées avec Facebook
    • Aucune action - Aucune action n'est prise
  3. Cliquez sur Next (Suivant).
  4. Confirmez les paramètres de synchronisation automatique, puis cliquez sur Terminer.

Configuration des règles Twitter

  1. Sélectionnez ce qui se passe lorsqu'une vidéo n'est pas balisée ou supprimée de Video Cloud :
    • Supprimer la vidéo distribuée sur Twitter - La vidéo sera supprimée de Twitter
    • Aucune action - Rien n'arrive à la vidéo Twitter
  2. Cliquez sur Next (Suivant).
  3. Confirmez les paramètres de synchronisation automatique, puis cliquez sur Terminer.

Affichage et modification des travaux de synchronisation automatique

Pour afficher et modifier les tâches de synchronisation automatique créées, procédez comme suit :

  1. Ouvrez Brightcove Social.
  2. Cliquez sur Synchronisation automatique dans la navigation de gauche. Une liste de toutes les tâches de synchronisation automatique qui ont été créées s'affiche.
    • Cliquez sur les liens dans la colonne Balises pour ouvrir le module Media et rechercher en fonction de ces balises.
    • Cliquez sur la colonne Destination pour ouvrir les paramètres de destination de publication.
    • Cliquez sur Supprimer pour supprimer la tâche de synchronisation.
  3. Cliquez sur le lien dans le nom de colonne Nom et ID de synchronisation automatique pour ouvrir les propriétés de synchronisation.
  4. Le STATUS peut être défini sur :
    • ACTIVE - Les vidéos seront synchronisées
    • PAUSE - Aucune vidéo ne sera synchronisée
  5. La liste SHOW peut être utilisée pour filtrer ce qui est affiché.
  6. Cliquez sur Paramètres pour modifier les paramètres de synchronisation. Cliquez sur Mettre à jour la sauvegarde des modifications.
  7. Cliquez sur Supprimer pour supprimer la tâche de synchronisation.
