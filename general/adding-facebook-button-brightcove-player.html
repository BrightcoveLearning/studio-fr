Facebook offre la possibilité d'ajouter un bouton J'aime à côté ou en dessous d'un lectuer. Cliquer sur le bouton J'aime affiche un lien vers la page Web hébergeant le lecteur sur la page Facebook du spectateur et ajoute une personne au nombre total de ceux qui ont aimé la vidéo. Il existe des options pour enregistrer le nombre total de « J'aime » et pour afficher les noms et les photos de profil des amis d'un utilisateur de Facebook qui a aimé une vidéo.

Création d'un bouton J'aime Facebook

Pour créer un bouton J'aime Facebook, procédez comme suit :

  1. Rendez-vous sur le site Facebook Developers: http://developers.facebook.com/docs/reference/plugins/like/.
  2. Spécifiez les préférences de votre bouton « J'aime » dans le formulaire de configuration comme indiqué sur la page. Dans le URL à aimer case, entrez l'URL de la page hébergeant votre lecteur.
  3. Cliquez sur Get Code (Obtenir le code).
  4. Cliquez sur iFrame en haut de la boîte de dialogue. Copiez le code.
  5. Ouvrez la page Web hébergeant le lecteur et collez le code sous le code de publication du lecteur. Ajouter un <br/> au-dessus du code Facebook pour afficher le bouton J'aime sous le lecteur.

Lorsque la page est publiée, les photos de profil de tous les amis du spectateur qui ont aimé la vidéo seront affichées. Les images varient selon l'utilisateur. Pour implémenter cette option, cochez Afficher les amis' Visages lors de la configuration des préférences du bouton J'aime.

Cette solution fonctionne au niveau de la page Web, pas au niveau de la vidéo, et ne prend en charge qu'un seul bouton Like par page. Les utilisateurs de Facebook aiment le lectuer. Si un lecteur contient une liste de lecture, l'option J'aime ne spécifie pas de vidéo individuelle. Pour publier un statut pour une vidéo individuelle, vos utilisateurs peuvent la partager à l'aide de la fonctionnalité de partage social.
