Renommer le module d'audience

14 juil. 2021

L'audience a été renommée Campagne. Cette mise à jour des fonctionnalités inclut également de nouvelles fonctionnalités, notamment :

  • Campaign Chrome Extension - Un nouvel affichage tête haute qui vous permet d'accéder en un coup d'œil aux performances vidéo directement à partir des pages sur lesquelles les vidéos sont affichées
  • Analyse comparative des vidéos: comparez les performances de vos vidéos à d'autres vidéos en fonction du type de contenu vidéo

Nouveau partenaire : Création de contenu Wibbitz

13 juil. 2021

Présentation du nouveau partenariat de Brightcove avec Wibbitz, conçu pour vous permettre de créer plus facilement le contenu opportun et pertinent que votre public souhaite, sur les chaînes qu'il visite déjà. Apprendre encore plus.

Vous pouvez utiliser la fonction de superposition d'images statiques pour superposer une image statique sur une vidéo dans un canal Cloud Playout.

Livraison en Chine

Pour plus de détails, consultez la présentation: Document de livraison en Chine.

Prise en charge des modules dynamiques pour les sorties RTMPS

16 avr. 2019

Le module Live a ajouté la prise en charge des sorties RTMPS. Les sorties RTMP permettent d'envoyer le flux en direct à d'autres points d'entrée RTMP, par exemple, en envoyant le live pour diffuser sur Facebook ou YouTube. Facebook nécessite spécifiquement des URL RTMPS. Pour plus d'informations, consultez Utilisation des sorties RTMP en direct pour diffuser sur Facebook et YouTube.

Nouvelle option de paramètres publicitaires côté serveur dans l'interface utilisateur d'administration

Pour plus d'informations, consultez Configuration des paramètres publicitaires côté serveur.

Options d'entrée et de sortie supplémentaires dans le module Live

30 janv. 2019

Le module Live a ajouté la prise en charge des formats d'entrée RTP et SRT (en plus de RTMP). La prise en charge des sorties RTMP a également été ajoutée. Les sorties RTMP permettent d'envoyer le flux en direct vers d'autres points d'entrée RTMP. Cela peut être utilisé pour envoyer le live à diffuser sur Facebook ou YouTube par exemple. Pour plus d'informations, voir Création et gestion d'événements en direct à l'aide du module Live.

Localisation du lecteur dans le module lectuers

Le module Players prend désormais en charge la localisation des lectuers, ce qui permet aux éléments de l'interface du lecteur d'apparaître dans différentes langues. Si la localisation du lecteur n'est pas configurée, la face inter/players/configuring-player-localization.html/players/configuring-player-localization.htmlface s'affiche en anglais. Pour plus d'informations, consultez Configuration de la localisation du lecteur.

  • La page de liste des vidéos prend désormais en charge le défilement horizontal

Mises à jour du module lectuers

30 nov 2018

Plusieurs nouvelles fonctionnalités ont été ajoutées à l'interface utilisateur du module Players :

  • Lors de l'ajout d'une superposition de logo, une URL de logo peut désormais être ajoutée. Cela ouvrira un nouvel onglet de navigateur lorsque vous cliquez sur la superposition.
  • Lorsque vous utilisez un lecteur de liste de lecture, la liste de lecture peut désormais être masquée. Ceci est utile lorsque vous voulez lire une série de vidéos sans que la liste de lecture soit affichée.
  • Un éditeur JSON est maintenant disponible pour modifier la configuration du lecteur.

Pour plus d'informations sur toutes les propriétés du lecteur, consultez Définition des propriétés du lecteur.

Lecture hors connexion, propriété vidéo

24 mai 2018

Une propriété vidéo de lecture hors connexion a été ajoutée dans les modules Upload et Media. Lorsque cette option est cochée, permet d'enregistrer la vidéo localement et de la visualiser hors connexion lors de l'utilisation d'applications créées à l'aide des kits SDK du périphérique. Cette option n'apparaît que si le compte est activé pour la lecture hors ligne.

Le module Upload peut désormais être configuré de sorte que tous les champs obligatoires doivent avoir des valeurs avant que les vidéos puissent être téléchargées. Pour plus d'informations, consultez Gestion des paramètres de compte.

Interface utilisateur mise à jour du module des lectuers -

12 mars 2018

Un nouvel éditeur de lectuer (la page des détails du lectuer) est maintenant disponible comme dans la version bêta opt-in. Les éditeurs verront un bouton sur la page des détails du lectuer en cours indiquant « Passer à l'éditeur bêta ». Lorsque vous cliquez dessus, un modal apparaîtra pour expliquer ce qu'est le nouvel éditeur de lectuer et comment les commentaires peuvent être soumis. La nouvelle et l'ancienne version de l'interface utilisateur peuvent être utilisées. Le commutateur est spécifique à l'utilisateur, de sorte que vous n'aurez pas d'impact sur l'interface utilisateur de tous les utilisateurs du compte lorsque vous choisissez d'opt-in. Notez que la possibilité de gérer les plugins de lectuer n'est pas encore disponible dans la nouvelle interface utilisateur.
