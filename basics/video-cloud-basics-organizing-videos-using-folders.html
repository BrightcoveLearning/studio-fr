Étapes

  1. Ouvrez le module Médias.
  2. Dans le panneau de navigation latéral, cliquez sur l'icône Créer le dossier lien.
  3. Saisissez un nom pour le dossier, puis appuyez sur entrer.
  4. Pour ajouter des vidéos à un dossier, sélectionnez une ou plusieurs vidéos.
  5. Cliquez sur Plus > Déplacer vers le dossier.
  6. Cliquez sur un dossier puis cliquez sur Bouge toi.

Tous les dossiers apparaîtront dans la navigation de gauche. Le nombre après le nom du dossier est le nombre de vidéos dans le dossier.

Pour supprimer des vidéos d'un dossier, cliquez sur un dossier, sélectionnez une ou plusieurs vidéos puis cliquez sur Supprimer des vidéos> Supprimer des vidéos du dossier.
