Étapes

  1. Dans Video Cloud Studio, ouvrez le Analytique module. Le rapport Performances s'affiche.
  2. Pour spécifier la période du rapport, utilisez le sélecteur de date en haut de la page.
  3. Sélectionnez pour afficher le Performance par Vidéo dimension.
  4. Clique sur le Vues vidéo colonne pour trier les données.

Le champ de recherche en haut de la page peut être utilisé pour rechercher une vidéo spécifique.
