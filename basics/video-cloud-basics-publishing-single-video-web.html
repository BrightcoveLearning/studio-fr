Conditions préalables

  • Vous avez téléchargé des vidéos sur votre compte Video Cloud
  • Vous avez a créé un lectuer

Étapes

  1. Ouvrez le module Médias.
  2. Sélectionnez une vidéo à publier en cochant la case en regard de la miniature de la vidéo.
  3. Cliquez sur Publier et incorporer> Publier sur le Web.
  4. Dans le Sélectionner un lectuer liste déroulante, sélectionnez un lecteur à utiliser.
    Note: Le lecteur par défaut de Brightcove est créé par défaut dans tous les comptes Video Cloud.
  5. Cliquez sur l' URL située sous le lecteur de prévisualisation pour l'ouvrir dans un nouvel onglet de navigateur. Confirmez que la vidéo s'affiche et qu'elle peut être lue.
  6. Copiez le code d'intégration et collez-le dans votre page Web.
