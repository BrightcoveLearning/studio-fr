Étapes

  1. Ouvrez le module Médias.
  2. Cliquez sur un titre vidéo pour ouvrir la page des propriétés de la vidéo.
  3. Utilisez le lecteur d'aperçu en haut de la page pour lire la vidéo.
  4. Mettez la vidéo en pause lorsque vous arrivez à l'image que vous souhaitez utiliser pour l'affiche/image miniature.
  5. Cliquez sur Capture.
  6. Sélectionnez cette option pour utiliser l'image comme affiche et/ou miniature.
  7. Cliquez sur Save.

Notez qu'il peut prendre une minute pour que le traitement se termine et que les images apparaissent dans le module Media.
