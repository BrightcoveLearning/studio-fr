Étapes

  1. Dans Video Cloud Studio, cliquez sur lectuers pour ouvrir le lectuers module.
  2. Cliquez sur New Player.
  3. Entrez un nom et une brève description.
  4. Cliquez sur Save.

Pour afficher la page des propriétés du lecteur, cliquez sur un nom de lecteur.
