Étapes

  1. Ouvrez le module Players.
  2. Cliquez sur New Player.
  3. Entrez un nom et une brève description.
  4. Cliquez sur Save.
  5. Cliquez sur le nom du lecteur pour ouvrir ses propriétés.
  6. Sélectionner un Type de lectuer de Playlist et cliquez sauver.
  7. Cliquez sur Style dans le menu de navigation de gauche.
  8. Sélectionnez un type de liste de lecture:
    • Playlist (Vertical) - La liste de lecture s'affiche sur le côté droit du lecteur.
    • Liste de lecture (horizontale) - La liste de lecture s'affiche en bas du lecteur.
    • Playlist (Hidden) - Aucune liste de lecture n'est affichée.
  9. Lorsqu'un Type de lectuer de Playlist est sélectionné, supplémentaire Relecture les propriétés peuvent être configurées. Cliquez sur Lecture dans le menu de navigation de gauche :
    • Ordre de lecture aléatoire - Les vidéos seront lues dans un ordre aléatoire
    • Lecture de la vidéo lorsque cette option est sélectionnée: si vous cliquez sur une miniature dans le menu de la liste de lecture, la vidéo sera lue lorsqu'elle est chargée, même si la vidéo précédente du lecteur a été suspendue
    • Mode de lecture continue - Les vidéos progressent automatiquement et se répètent indéfiniment
    • Compte à rebours vidéo: uniquement activé lorsque le mode de lecture continue est coché
      • Carte suivante (avance automatique) - Une carte suivante est affichée dans le lectuer
      • Écran de fin suivant (compte à rebours) - Un compte à rebours d'écran de fin s'affiche dans le lecteur
  10. Cliquez sur Save pour enregistrer les modifications apportées.
  11. Pour publier les modifications sur le lecteur, cliquez sur Publier et intégrer et alors Publier les modifications.
